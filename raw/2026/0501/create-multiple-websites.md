# Create multiple websites

You can conveniently manage multiple websites under one login. This guide will show you how to create a new site in your WordPress.com account and switch between your different sites.

## Create Your First Site

When you sign up with WordPress.com, we'll automatically create a new site for you.

If you don't already have a WordPress.com account, fill out [this signup form](https://wordpress.com/start) to provide an email address, a username, and password.

In the next steps, you'll be asked for your website address (known as a domain.) You can create a new domain, connect a domain you already own, or continue with a free .wordpress.com site address.

Lastly, you'll select a [WordPress.com plan](https://wordpress.com/plans/) to set up your new website. Each plan applies to one site only.

## Add a New Site

To create a new site in your existing account, follow these steps:

1. [Log in](https://wordpress.com/log-in) to your account.
2. From your site's dashboard, click the W icon in the top-left to navigate to [your hosting dashboard](https://wordpress.com/sites).
3. Click the "Add new site" button in the upper-right corner and choose from:

- **WordPress.com:** create a fresh new site in your WordPress.com account.
- **Build with AI:** Prompt, edit, and launch a WordPress website with the power of AI.
- **Via the Jetpack plugin:** Add the power of WordPress.com to a self-hosted WordPress website.
- **Migrate:** Bring a website onto WordPress.com from any hosting provider.
- **Import:** Import a content-only file from any WordPress site.

4. When prompted, select a domain and a WordPress.com plan for this new site to finish creating it.

Each site will be completely independent of each other, with different designs, domains, subscribers, and settings. A plan applies to one site only. One plan cannot be shared across multiple websites.

You can also [create a copy of an existing site](https://wordpress.com/support/copy-a-site/).

## Create a Staging Site

If you are planning to create a new site because you want to test plugins, updates, or changes, without impacting your production site, you can [create a staging site](https://wordpress.com/support/how-to-create-a-staging-site/) instead of creating a temporary second site. You can transfer the site from staging to production with the click of a button.

## Switch Between Sites

With multiple websites under your account, you can switch from one site to the other by taking these steps:

1. Visit any of your sites' dashboards.
2. Click the "W" logo in the upper-left corner to visit your [hosting dashboard](https://wordpress.com/sites).
3. Select the website you wish to work on.
4. Click the "My Home" or "WP Admin" link to be taken to that site's dashboard.

## Move a Domain to a New Site

If you have added a new domain name on your existing site but intended to create a new site for that domain, you can move the domain to a new site:

1. Visit the [hosting dashboard](https://wordpress.com/sites) and click the "Add new site" button.
2. If your domain is registered in your WordPress.com account, type your domain into the box and select it from the results.

If your domain is not registered at WordPress.com, do not enter your domain at this stage. Select the free .wordpress.com address for now.

If your domain is not yet added to the new site (because it was connected from another provider), follow the steps to [Move a Domain to Another WordPress.com Site](https://wordpress.com/support/domains/move-a-domain-to-another-wordpress-com-site/).

## Using a Plan for a New Site

A paid hosting plan is applied to the site you made the purchase for. You have a few options for starting a new site using the same hosting plan:

- If you want to start a new site under the same plan, you can [register a new domain](https://wordpress.com/support/domains/register-domain/) and [clear the content](https://wordpress.com/support/reset-your-site/) for the current site.
- If you need to transfer a paid plan from one website to another site on your account, [contact support](https://wordpress.com/support/contact/) for assistance. Only the current plan owner can request to move the plan to a new site or person.
- If you wish to move a plan to another user account, you can [transfer the site itself](https://wordpress.com/support/transfer-a-site/) and the plan will move with it.
