# Use the WP Admin dashboard

The WP Admin dashboard is an administrative interface for managing your WordPress sites. This guide will show you how to access the WP Admin dashboard for your WordPress.com site.

## What is WP Admin?

WP Admin, also known as /wp-admin or the classic WordPress dashboard, is the original administration interface for managing a WordPress website. It's sometimes called the WordPress dashboard, the admin area, or simply "the backend."

## Access WP Admin

To access WP Admin for any of your WordPress.com sites:

1. Visit your Sites list: [https://wordpress.com/sites](https://wordpress.com/sites)
2. Hover your mouse over one of your websites.
3. Click the WP Admin link that appears.

![WP Admin access link](https://en.support.wordpress.com/wp-content/uploads/2026/03/wp-admin-access-link.jpg)

You can also access WP Admin by typing your website address in your browser's address bar, followed by `/wp-admin`. For example:
- `yourgroovysite.wordpress.com/wp-admin`
- `yourgroovydomain.com/wp-admin`

If you are logged into WordPress.com but unable to log into the WP Admin dashboard, follow the [steps to reset your WP Admin password](https://wordpress.com/support/wordpress-com-secure-sign-on-sso/#prompted-to-log-in-again-can-t-remember-wp-admin-password).

## Set Default Interface Style

You can set your website to use the WP Admin interface (Classic style) whenever you log into your site's dashboard. Learn how to set your site's admin interface to [always use WP Admin](https://wordpress.com/support/set-your-admin-interface-style/#choose-your-interface-style).

## WP Admin Dashboard Home Screen

Regardless of what you have set as your [default interface style](https://wordpress.com/support/set-your-admin-interface-style/#choose-your-interface-style), you can access the WP Admin dashboard home screen.

To access the WP Admin dashboard home screen:

1. Visit your [site's dashboard](https://wordpress.com/home/).
2. Click the Dashboard link at the top of the left side menu.

The dashboard home screen displays cards of information, including:

- **At a Glance:** a summary of the content on your site and identifies which theme you are using.
- **Activity:** the upcoming scheduled posts, recently published posts, and the most recent comments on your posts.
- **Quick Draft:** quickly create a new post and save it as a draft. It also displays links to the three most recent draft posts you've started.
- **Site Health Status** *(plugin-enabled sites only):* displays information about your WordPress configuration and items that may need your attention.

You'll also find cards for other features, including [Stats](https://wordpress.com/support/stats/), [Newsletter](https://wordpress.com/support/newsletter/), [Writing Prompts](https://wordpress.com/support/writing-prompts/), and [any cards added by plugins](https://wordpress.com/support/plugins/) you have installed.

Click on any of the three arrows next to the title in any box to move the current box element up, down, or toggle to shut it. You can also click and drag any boxes to move them into a different position.

## Screen Options

All core pages in WP Admin include an option to choose which columns display, how many items to show (pagination), and whether to show a compact or extended view. These screen options are especially useful if you're viewing your tools from a smaller screen and want to be able to see specific elements at a glance.

Learn how [to customize your administration screens](https://wordpress.com/support/customize-your-administration-screens/) with Screen Options.
