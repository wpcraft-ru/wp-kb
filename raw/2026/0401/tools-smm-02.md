## Фокус на Social Listening

### Buffer
- **Scheduling и публикация** — управление контентом через Instagram, Facebook, LinkedIn, Twitter, Pinterest
- **Базовая аналитика** — посты, engagement, рост аудитории
- **Team collaboration** — approval workflows, общие календари
- **AI Assistant** — только для генерации идей контента, НЕ для аналитики
- **Social Listening:** НЕТ

Вывод: Buffer — инструмент для публикации, не для мониторинга.

### Hootsuite
- **Bulk scheduling** — массовая публикация по расписанию
- **Интегрированная аналитика** — engagement, демография, performance
- **Content Discovery** — поиск трендового контента
- **150+ интеграций** — CRM, Canva, Google Suite
- **Social Listening (Talkwalker):** AI-powered мониторинг упоминаний, хэштегов, тональности, трендов

### Сравнение Social Listening

| Функция | Buffer | Hootsuite |
|---------|--------|-----------|
| Audience insights | Базовые метрики (engagement, reach, рост) | Углублённые (демография, тональность, Brandwatch) |
| AI-powered insights | Нет | Quick Search, trending hashtags, тональность |
| Brand health tracking | Ограничен | Hootsuite Listening + Talkwalker, анализ тональности |
| Competitor benchmarking | Нет | До 20 конкурентов (Team) |
| Sentiment analysis | Нет | Есть (положительная/отрицательная/нейтральная) |

### Ключевые выводы
- Hootsuite значительно сильнее в social listening и brand health tracking
- Buffer — простой и дешёвый инструмент для публикации и базовой аналитики
- Для серьёзного social listening Hootsuite с Talkwalker — минимальный порог входа
- Gartner: 68% приобрели инструмент social monitoring, но только 19% используют его полностью
