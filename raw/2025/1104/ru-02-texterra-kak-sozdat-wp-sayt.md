# Как создать сайт на WordPress с нуля: полное руководство для новичков
> Источник: https://texterra.ru/blog/kak-sozdat-sayt-na-wordpress-polnoe-rukovodstvo-dlya-novichkov.html

WordPress — самая распространённая CMS: ~64.8% всех сайтов используют WordPress.

## Почему WordPress
- Бесплатный движок (GNU GPL)
- Неограниченные возможности: блог, магазин, портал
- Гибкая настройка: темы и плагины
- Простота администрирования
- Автоустановка на хостинг

## Шаг 1: Выбор хостинга и регистрация домена
Рекомендованные провайдеры: Reg.ru, Beget, Netangels.

### Выбор доменного имени
- Совпадение с названием сайта
- Максимально короткое
- Легко запоминается
- Домены верхнего уровня: .com, .info, .org, .net, .ru, .club, .guru и др.

## Шаг 2: Установка WordPress
- Скачать с ru.wordpress.org
- Загрузить через FTP (FileZilla)
- Создать БД, указать данные подключения
- Указать название сайта, логин/пароль администратора

## Шаг 3: Выбор темы
- Бесплатные из официального каталога (wordpress.org/themes)
- Платные от одобренных разработчиков
- Критерии: адаптивный дизайн, не перегружена функционалом, частота обновлений

### Русификация темы
- Вручную (не рекомендуется)
- Плагин Loco Translate
- Через сторонние компании
- Программа Poedit

## Шаг 4: Настройка CMS
- Общие: название, описание, URL, часовой пояс, язык
- Написание: формат записей, рубрики, сервисы обновления
- Чтение: главная страница, количество записей, видимость для поисковиков
- Обсуждение: модерация, чёрный список
- Постоянные ссылки: структура URL

## Шаг 5: Плагины
### Безопасность
- WP Database Backup
- BackUpWordPress
- Backup and Restore WordPress

Рекомендация: минимум плагинов, только необходимые. Каждый лишний плагин замедляет сайт.
