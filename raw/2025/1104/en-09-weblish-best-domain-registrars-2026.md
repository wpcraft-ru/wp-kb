## Selection Criteria
- Pricing transparency (30% weight)
- DNS speed (25% weight)
- Security features (20% weight)
- Support response (15% weight)
- User reviews (10% weight)

## DNS Performance
- SLA with <0.01% downtime
- Global anycast network
- DNSSEC, DoH/DoT support
- Updates propagate in minutes
- Health checks and alerts

## Migration Checklist
1. Audit domains, contacts, nameservers
2. Lower TTL to 300s 24h before
3. Unlock domains, get auth codes
4. Initiate transfer (5-7 days)
5. Replicate DNS records exactly
6. Monitor for 48h after switch
