# ⚡ Производительность и CDN — материалы

## 🌍 Cloudflare

**Тип:** Глобальный CDN + безопасность + оптимизация
**Сайт:** cloudflare.com

### Ключевые факты (2025)
- 250-310+ edge-локаций по миру
- Бесплатный тариф: SSL, базовый DDoS, кэширование статики (без HTML)
- APO (Automatic Platform Optimization): кэширует HTML на edge, снижает TTFB на 70%+ (платные тарифы)
- Официальный WordPress-плагин: авто-интеграция DNS, кэширования, сброс кэша при обновлении контента
- Argo Smart Routing: оптимизация маршрутов (Business/Enterprise)
- Core Web Vitals: измеряемый рост Lighthouse/PageSpeed scores
- Подходит для: сайтов с глобальным трафиком, визуальным контентом, всплесками трафика
- Ограничения: на медленных origin-серверах может не дать прироста; бесплатный план не кэширует HTML

### Цены
- Free: базовый CDN + SSL + DDoS
- Pro: ~$20/мес (APO, Polish image optimization)
- Business: ~$200/мес (Argo, 100% uptime SLA)
- Enterprise: кастом

### Интеграция с WordPress
- Официальный плагин Cloudflare
- Авто-сброс кэша при публикации/обновлении
- Рекомендуется использовать вместе с кэширующим плагином (WP Rocket, LiteSpeed Cache)

## 🌍 Bunny.net

**Тип:** CDN + хранилище + оптимизация
**Сайт:** bunny.net

### Ключевые факты (2025)
- 100-120 PoP по миру
- Pay-as-you-go: ~$0.01/GB в большинстве регионов
- 5 TB трафика EU/NA ≈ $50 (против ~$425 на AWS CloudFront)
- Выделенный плагин «Bunny WordPress CDN»
- Интеграция с WP Rocket и другими кэширующими плагинами
- Бесплатный TLS/SSL, DDoS-защита
- Хорошая производительность в Европе

### Плюсы
- Очень доступная цена для малых/средних сайтов
- Простая настройка (DNS + плагин = минуты)
- Хорошая скорость в Европе

### Минусы
- Нет full-page кэширования на уровне CDN (нужен отдельный плагин для HTML)
- Меньше PoP чем у Cloudflare/AWS
- Нет постоянного бесплатного тарифа (только триал ~14 дней)

### Для кого
- Блоги, портфолио, small business сайты
- Те, кому нужен дешёвый CDN для статики + свой кэширующий плагин

## 🌍 NitroPack

**Тип:** All-in-one оптимизация (кэширование + CDN + оптимизация)
**Сайт:** nitropack.io

### Ключевые факты (2025)
- Облачный сервис: кэширование, оптимизация изображений, минификация, lazy-loading, шрифты, встроенный CDN
- 4.9★ Trustpilot, 4.2M+ загрузок WordPress
- Core Web Vitals pass rate: ~52% (выше чем WP Rocket ~48%, WP-Optimize ~47%)
- Типичное снижение LCP: с нескольких секунд до <2 сек
- PageSpeed scores: часто 90+ сразу после активации
- Поддержка WooCommerce, page builders, большинства тем

### Плюсы
- Set-and-forget: автоматическая оптимизация HTML, CSS, JS, изображений, шрифтов
- Заменяет: кэш-плагин + оптимизатор изображений + CDN
- Очень быстрый результат "из коробки"

### Минусы
- Нет оптимизации базы данных
- Зависимость от облака NitroPack (vendor lock-in)
- Жалобы на «feature-locking» — новые фичи только на дорогих тарифах
- Некоторые функции требуют верхних тарифов

### Цены
- Бесплатный: 5,000 pageviews/мес, 1 сайт
- Business: ~$21/мес, 50,000 pageviews, 1 сайт
- Growth: ~$51/мес, 200,000 pageviews, 1 сайт
- Scale: ~$176/мес, 1M pageviews, 1 сайт

## 🇷🇺 Yandex Cloud CDN

**Тип:** Российский CDN + облачная инфраструктура
**Сайт:** cloud.yandex.ru

### Ключевые факты (2025)
- 37 PoP-точек в России + локации в СНГ
- Отличные TTFB и задержки для РФ/СНГ аудитории
- Сегментация и параллельная загрузка больших файлов
- Гибкая настройка TTL, GZIP-сжатие
- Не требует специфичного WordPress-плагина (CNAME + настройки кэша)

### WordPress-интеграция
- HTML: через бэкенд, TTL 30-60 минут
- Статика: через CDN-домен (CNAME), TTL до 7 дней, GZIP
- Предзагрузка больших файлов
- Нет «умного» HTML-кэша как у Cloudflare APO

### Плюсы
- Отлично для РФ/СНГ: низкие задержки, высокая пропускная способность
- Гибкая настройка кэширования
- Интеграция с экосистемой Yandex Cloud (Object Storage, безопасность)

### Минусы
- Меньше PoP для глобальной аудитории
- Нет APO-подобного HTML-кэширования

### Для кого
- Сайты с аудиторией РФ, Казахстан, Узбекистан
- Медиа-блоги, новостные ресурсы, каталоги

## 🇷🇺 Selectel CDN

**Тип:** Российский CDN + облачные серверы + S3-хранилище
**Сайт:** selectel.ru

### Ключевые факты (2025)
- Облачные серверы + объектное хранилище (S3-совместимое) + CDN-сеть
- Плагин Selectel Storage Upload для WordPress
- Синхронизация медиафайлов в облако Selectel
- Возможность хранить файлы только в облаке (без дублирования на сервере)
- Интеграция с Akamai CDN

### WordPress-интеграция
- Плагин Selectel Storage Upload
- Альтернативно: универсальные S3-плагины (Media Cloud)
- Настройка через CNAME для CDN-домена

### Плюсы
- Ускорение через CDN для статики
- Снижение нагрузки на веб-сервер
- Отказоустойчивость (бэкап медиа в облаке)
- Гибкость масштабирования

### Минусы
- Сложнее старт чем на обычном хостинге
- Требует настройки сервера, PHP, базы, SSL, DNS, CDN
- Тарификация CDN-трафика может быть выше базового

### Типичная схема
1. Облачный сервер Selectel → WordPress
2. Контейнер Selectel Storage → синхронизация медиа
3. CDN-ресурс → раздача статики через CNAME

## 🇷🇺 VK Cloud CDN

**Тип:** Российская облачная инфраструктура + CDN
**Сайт:** cloud.vk.com

### Ключевые факты (2025)
- Готовый WordPress в «магазине приложений» (LAMP/LEMP)
- Собственная CDN-сеть VK
- S3-совместимое хранилище (VK Object Storage)
- Бэкапы на S3, стандартные плагины и темы
- HTTP/3, DDoS-защита

### WordPress-интеграция
- CDN через плагины кэширования (W3 Total Cache, WP Super Cache)
- Настройка «Generic Mirror» с доменом CDN-ресурса
- Источник (origin): WordPress-сервер или бакет

### Плюсы
- Российская инфраструктура, низкие задержки для РФ/СНГ
- Гибкое масштабирование VPS/CDN
- HTTP/3, DDoS-защита

### Минусы
- Нужен базовый уровень Linux-администрирования
- Стоимость = VPS + трафик CDN (может быть дороже shared-хостинга)
- Не для новичков, которым нужно «одна кнопка»

### Для кого
- Средние WordPress-сайты, интернет-магазины с аудиторией РФ/СНГ
- Проекты, где важен контроль над инфраструктурой
