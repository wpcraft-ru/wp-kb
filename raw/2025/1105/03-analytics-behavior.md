# 📊 Аналитика и поведение пользователей — материалы

## 🇷🇺 Yandex Metrica

**Тип:** Российская веб-аналитика (бесплатно)
**Сайт:** metrika.yandex.ru

### Ключевые факты (2025)
- Полноценная замена Google Analytics для РФ/СНГ
- Бесплатный сервис: посещаемость, поведение, конверсии, источники трафика
- Официальный плагин для WordPress: **Yandex Metrica** / **WP Yandex Metrica**

### WordPress-интеграция
- Авто-установка счётчика без кода
- **E-commerce отслеживание без кода:** просмотр товара, корзина, оформление заказа
- Дашборд-виджет в админке WordPress (графики, топ-страницы)
- Настройка по ролям пользователей
- Click-карта и Session Replay (вебвизор)
- API-режим и базовый режим

### Обновления 2025
- Новый интерфейс отчётов (группировки, фильтры, условия)
- Отчёты по часам/дням недели/месяцам
- Офлайн-конверсии и конверсии из мессенджеров (WhatsApp, Telegram, Viber)

### Для кого
- Блоги, интернет-магазины, корпоративные сайты на WordPress с аудиторией РФ

## 🇷🇺 Roistat

**Тип:** Сквозная аналитика + коллтрекинг + управление рекламой
**Сайт:** roistat.com

### Ключевые факты (2025)
- Связывает: посетитель сайта → заявка → звонок → сделка → прибыль
- Цепочка: «реклама → сайт → лид → продажа → ROI»
- Интеграции: WooCommerce, Contact Form 7, Gravity Forms
- AI-прогноз по продажам, модели атрибуции
- Авто-отключение убыточных рекламных кампаний

### WordPress-интеграция
- Готовые интеграции с WooCommerce, CF7, Gravity Forms
- API-коннекторы (ApiMonster и др.) для сложных связок
- JS-скрипт для кастомных событий

### Плюсы
- Единый дашборд: сайт + CRM + 1С + реклама
- ROI по каждому каналу
- AI-прогнозы и автоматизация

### Минусы
- Высокий порог входа по цене
- Сложная настройка для больших WooCommerce
- Российский фокус (Яндекс.Директ, российские CRM)

## 🇷🇺 Calltouch

**Тип:** Коллтрекинг + сквозная аналитика
**Сайт:** calltouch.ru

- Связка звонков с визитами и формами

- Плагин Calltouch (авто-добавление счётчика и скрипта коллтрекинга)

### Плюсы
- Готовый плагин для WordPress
- Сильная аналитика по звонкам
- Интеграция с популярными CRM

### Минусы
- Платная модель (тарифы по номерам, трафику, доменам)
- Сложные кастомные интеграции требуют API/вебхуки

## 🌍 Google Analytics 4 (GA4)

**Тип:** Глобальная веб-аналитика (условно бесплатно)
**Сайт:** analytics.google.com

### Ключевые факты (2025)
- Event-based tracking вместо pageviews
- Основные WordPress-плагины для интеграции:
  - **MonsterInsights/ExactMetrics:** полный eCommerce, дашборды в WP, GDPR
  - **GA4WP:** лёгкий дашборд, IP-анонимизация, Ads + Optimize
  - **HT Easy GA4:** WooCommerce + Google Ads events, video/audio engagement
  - **Site Kit by Google:** минимальная глубина GA4, но Search Console + AdSense
  - **AnalyticsWP:** GDPR-фокус, per-page stats

### Функции WordPress-плагинов GA4
- Авто-вставка gtag.js или GTM
- Realtime/near-realtime статистика
- Авто-трекинг: outbound links, downloads, scroll depth, 404
- WooCommerce enhanced eCommerce
- Дашборды внутри админки WordPress
- Cookie consent, IP anonymization
- Интеграция с Google Ads, Meta Pixel, TikTok Pixel

## 🌍 Hotjar

**Тип:** Heatmaps + session recordings + feedback
**Сайт:** hotjar.com

### Ключевые факты (2025)
- Heatmaps (клики, скролл, движение мыши)
- Session recordings с AI-тегированием (rage clicks, form abandonment)
- On-page surveys и feedback widgets
- Официальный WordPress-плагин
- Сегментация по устройствам, источникам трафика

### Плюсы
- Наглядные визуальные отчёты
- AI-ассистент для поиска UX-проблем
- Всё в одном: heatmaps + записи + опросы

### Минусы
- Может замедлять сайт при большом количестве других JS-инструментов
- Бесплатный тариф: 300 записей, 3 heatmaps (мало для активных сайтов)
- Платные тарифы быстро дорожают с ростом трафика
- Жалобы на inconsistent support и задержки обновлений плагина

## 🌍 Microsoft Clarity

**Тип:** Бесплатные heatmaps + session recordings (Microsoft)
**Сайт:** clarity.microsoft.com

### Ключевые факты (2025)
- **Полностью бесплатно** (без ограничений по трафику)
- Session recordings с AI-суммаризацией (Clarity Copilot)
- Heatmaps, scroll-depth, rage-click, dead-click detection
- Официальный WordPress-плагин: дашборд внутри админки
- Авто-маскировка чувствительных данных (GDPR-friendly)
- Лёгкий клиентский скрипт, минимальное влияние на скорость
- С октября 2025 — cookie consent для EEA/UK/Switzerland

### Плюсы
- Бесплатно без ограничений
- AI-суммаризация (до 250 recordings → 1 отчёт)
- Идеально в паре с GA4: GA4 = «что», Clarity = «почему»

### Минусы
- Меньше функций чем Hotjar (нет surveys, нет feedback widgets)
- Требует cookie consent для EU/UK
