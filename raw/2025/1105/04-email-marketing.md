# ✉️ Email-маркетинг и коммуникации — материалы

## 🇷🇺 UniSender

**Тип:** Российский сервис email-рассылок
**Сайт:** unisender.com

### Ключевые факты (2025)
- Официальный плагин **UniSender Integration** для WordPress
- Формы подписки: сайдбар, страницы, посты, футер
- Авто-синхронизация контактов через API
- Поддержка дополнительных полей
- Привязка к разным спискам рассылок

### WordPress-интеграция
- Плагин → API-ключ → список рассылки
- Шорткоды и виджеты для форм

### Плюсы
- Простая интеграция «из коробки»
- Поддержка доп. полей
- Российский сервис, хорошая доставляемость по РФ

### Минусы
- Плагин только для сбора подписчиков, рассылки — в интерфейсе UniSender
- Для сложных CRM-сценариев нужны доп. плагины (Contact Form 7, Лейка)

### Цены
- Бесплатный тариф: ограничен
- Платные: от ~500 ₽/мес

## 🇷🇺 SendPulse

**Тип:** Мультиканальный маркетинг (email + SMS + web push + чат-боты)
**Сайт:** sendpulse.com

### Ключевые факты (2025)
- Официальный плагин **SendPulse Email Marketing Newsletter** для WordPress
- Бесплатный лимит: до 15 000 писем/мес
- Drag-and-drop редактор, сегментация, персонализация
- Импорт базы контактов из WordPress
- Мультиканал: email + SMS + веб-пуш + чат-боты в мессенджерах
- Встроенные лендинги

### WordPress-интеграция
- Плагин → API-ключи (ID/Secret) →几分钟 настройки
- Стилизация формы под бренд
- Аналитика: открытия, клики, отказы

### Плюсы
- Бесплатный тариф (до 15 000 писем)
- Мультиканальность
- Готовые шаблоны, конструктор писем
- Простая интеграция

### Минусы
- Интерфейс может быть перегружен
- Ранее были XSS-уязвимости в плагине (важно обновлять)
- При росте базы рассылки дорожают

### Для кого
- Блоги, малый бизнес, интернет-магазины на WordPress
- Те, кто уже используют SendPulse для SMS/пушей

## 🇷🇺 Carrot Quest

**Тип:** Чат-бот + email-рассылки + микро-CRM
**Сайт:** carrotquest.io

### Ключевые факты (2025)
- Гибрид: чат-бот + email-рассылки + простая CRM
- Плагин для WordPress + интеграция с WooCommerce (до 7.1.0)
- Конструктор сценариев: триггеры по поведению → автоматические сообщения
- Email-цепочки по триггерам (подписка, просмотр товара, корзина)
- A/B тестирование по доходу и конверсии
- Подключение мессенджеров: WhatsApp, Telegram, Viber

### Функции для WordPress
- Чат-окно + чат-боты (сбор email, заявок, ведение к оплате)
- Поведенческие поп-апы и формы подписки
- Все данные → единая карточка пользователя
- Триггерные email-воронки

### Плюсы
- Один сервис вместо чата + email + простой CRM
- Глубокая поведенческая персонализация
- A/B тесты по доходу

### Минусы
- Тарифы зависят от количества посетителей (может быть дорого при высоком трафике)
- Жалобы на авто-списание доплат

### Для кого
- WordPress + WooCommerce с потребностью в едином инструменте
- Онлайн-школы, сервисные компании

## 🌍 Mailchimp

**Тип:** Популярный email-маркетинг (глобальный)
**Сайт:** mailchimp.com

### Ключевые факты (2025)
- Интеграция через официальный плагин или **MC4WP (Mailchimp for WordPress)**
- Pop-ups, inline-формы, sidebar-виджеты
- Чекбоксы на комментариях, регистрации, WooCommerce/EDD checkout
- Автоматизация: welcome emails, теги, сегментация
- GDPR-инструменты, double opt-in

### Плюсы
- Простая настройка, много готовых шаблонов
- Хорошая доставляемость
- Огромная экосистема WordPress-плагинов
- Подходит блогам и small business

### Минусы
- Дорожает с ростом базы (pricing по контактам)
- Автоматизация слабее чем ActiveCampaign/Klaviyo
- Ограниченный support на низких тарифах

### Цены
- Free: до 500 контактов, ~1,000 писем/мес
- Essentials: от $13/мес
- Standard: от $20/мес
- Premium: от $350/мес

## 🌍 Brevo (бывший Sendinblue)

**Тип:** Email + SMS + чат (pay-per-email, не по контактам)
**Сайт:** brevo.com

### Ключевые факты (2025)
- Официальный WordPress-плагин: формы, SMTP, WooCommerce
- **Бесплатный тариф: 9,000 писем/мес, неограниченные контакты**
- Pay-per-email pricing (не по количеству контактов!)
- SMTP для транзакционных писем WordPress
- Drag-and-drop automation workflows
- Aura AI: subject lines, copy, send-time optimization
- Мультиканал: email + SMS + WhatsApp + web push
- WooCommerce: abandoned cart, post-purchase follow-ups

### Плюсы
- Выгодная модель оплаты (pay-per-email)
- Очень щедрый бесплатный тариф
- SMTP для WordPress-уведомлений
- Мультиканальность в одном интерфейсе

### Минусы
- Автоматизация слабее чем ActiveCampaign/Klaviyo
- Меньше нативных интеграций чем у Mailchimp (Zapier для остального)
- UI иногда глючит

## 🌍 HubSpot

**Тип:** CRM + маркетинг + sales (всё в одном)
**Сайт:** hubspot.com

### Ключевые факты (2025)
- Официальный WordPress-плагин: **HubSpot All-in-One Marketing**
- Бесплатный CRM
- Формы, popups, live chat, чат-боты → все данные в одном месте
- Email-кампании, workflows, списки
- Отчёты внутри WordPress-админки

### Плюсы
- Единая база: CRM + маркетинг + sales
- Inbound-маркетинг «из коробки»
- Бесплатный CRM

### Минусы
- Маркетинг-автоматизация дорого (платные Hubs)
- Vendor lock-in (привязка к HubSpot)
- Меньше plugin-ecosystem свободы vs чистый WordPress

### Для кого
- Компании, уже использующие HubSpot как CRM
- Small-mid teams с прямыми sales-процессами
