# 🧠 Автоматизация и интеграции — материалы

## 🇷🇺 Albato

**Тип:** Российская no-code платформа автоматизации
**Сайт:** albato.ru

### Ключевые факты (2025)
- 1 000+ сервисов (CRM, Google Sheets, WhatsApp, Telegram, чат-боты, календари)
- Принцип: триггер в WordPress → действие в другом сервисе
- Три способа подключения WordPress:
  - Application Password
  - Basic Auth
  - JWT-токен через плагин

### Типовые сценарии для WordPress
- Новый пользователь → CRM/Google Sheets
- Новая запись → рассылка в Telegram/WhatsApp
- Новый комментарий → уведомление в Slack/Telegram
- Заказы WooCommerce → CRM + уведомления
- Синхронизация WordPress ⇄ SaaS (Tidycal, Nifty, Odoo)

### Плюсы
- Простой визуальный конструктор «триггер-действие»
- Широкий набор интеграций + API/вебхуки
- Работа в реальном времени
- Российский сервис, поддержка на русском

### Минусы
- Нет официального нативного плагина Albato для WordPress
- Для сложных форм нужны доп. webhooks или код

## 🇷🇺 ApiMonster

**Тип:** No-code коннектор для WordPress (российский)
**Сайт:** apimonster.ru

### Ключевые факты (2025)
- Визуальный конструктор связок
- WordPress-события → API других сервисов
- Готовые бандлы: WordPress + Bitrix24, Poster, MPMonster, Контур.Школа, FileStorage
- Универсальные веб-хук-коннекторы
- 30 дней бесплатного теста

### Типовые сценарии
- Contact Form 7 / WooCommerce → лид в CRM
- WordPress → 1С, Bitrix24, Telegram, платежные системы, маркетплейсы
- WordPress + ApiMonster AI (генерация контента, обработка лидов)
- Сквозная аналитика и лид-трекинг

### Плюсы
- 5-10 минут на подключение
- Не нужен программист для типовых связок
- Сотни сервисов через универсальные коннекторы

### Минусы
- Сложные бизнес-логики всё равно требуют доработки
- Часть сценариев только на платных тарифах

## 🇷🇺 Make (ex-Integromat)

**Тип:** Визуальная no-code автоматизация (европейский сервис, популярен в RU)
**Сайт:** make.com

### Ключевые факты (2025)
- Официальный плагин **Make Connector** для WordPress
- Визуальный scenario builder (ветвление, фильтры, трансформации)
- 1 000+ сервисов
- Готовые шаблоны для WordPress

### WordPress-интеграция
- Плагин Make Connector → API-ключ → REST API endpoint
- Модули: посты, категории, пользователи, медиа, комментарии

### Типовые сценарии
- Google Sheets → ChatGPT → Make → авто-пост в WordPress
- Форма/лендинг → CRM → Make → пост/пользователь в WordPress
- WordPress как триггер: новый пост → рассылка, соцсети, Slack, отчёты
- Синхронизация WooCommerce с CRM/складами

### Плюсы
- Сложные многошаговые workflow
- Дешевле Zapier при масштабировании
- Мощный API и webhook support
- Бесплатный тариф (1,000 ops/мес)

### Минусы
- Более высокий порог входа чем Zapier
- Плагин иногда закрывают на review
- Overkill для простых задач

## 🌍 Zapier

**Тип:** Самая популярная no-code платформа автоматизации
**Сайт:** zapier.com

### Ключевые факты (2025)
- 5 000-6 000 подключённых приложений
- WordPress-триггеры: новые посты, пользователи, формы, WooCommerce-заказы
- Действия: CRM (HubSpot, Salesforce), email (Mailchimp), Slack, Google Sheets
- Готовые шаблоны (Zaps) для типовых WordPress-сценариев
- Плагин **Zapier for WordPress** для отправки триггеров

### Плюсы
- Очень простой интерфейс
- Огромная библиотека приложений
- Идеально для линейных workflow

### Минусы
- Дорожает с ростом задач (pricing per task)
- Только линейная логика (нет сложного ветвления)
- Внешний сервис, не внутри WordPress

### Цены
- Free: 100 задач/мес
- Starter: ~$20/мес (750 задач)
- Professional: ~$49/мес (2K задач)
- Team: ~$69/мес
- Company: от ~$139/мес

## 🌍 n8n

**Тип:** Open-source self-hosted автоматизация
**Сайт:** n8n.io

### Ключевые факты (2025)
- Open-source, self-hosted (можно хостить на своём сервере)
- Node-based визуальный редактор
- Официальный WordPress-node (CRUD: посты, страницы, пользователи)
- Сотни готовых нод и community templates

### WordPress-подключение
1. **Официальный WordPress node** — базовый CRUD через username + app password
2. **Webhooks через WordPress-плагины** (WP Webhooks, Bit Integrations)
3. **Прямые REST API calls** через HTTP Request node

### Типовые сценарии
- AI пишет черновик → n8n форматирует → авто-публикация в WordPress
- WooCommerce заказ → CRM + welcome email
- Форма → Slack + Google Sheets
- Контент-пайплайны: внешние данные → WordPress

### Плюсы
- Полный контроль и приватность (self-hosted)
- Очень гибкая логика (условия, циклы, code nodes)
- Выгодно при масштабировании (нет платы за tasks)
- Rich ecosystem нод и шаблонов

### Минусы
- Высокий порог входа (нужен технический специалист)
- Maintenance overhead (обновления, бэкапы, безопасность)
- WordPress node только базовый CRUD — сложное требует API calls

### Для кого
- Технические команды, агентства, compliance-sensitive проекты
- Те, кто хочет избежать vendor lock-in
