# Change Colors in the Customizer
Last reviewed on March 31, 2026
Copy page

## Theme Support
The settings described in this guide apply to [classic themes](https://wordpress.com/support/theme-types/#classic-themes) that support Custom Colors.
To find out if a theme has Custom Colors support, activate the theme and go to *Appearance → [Customize](https://wordpress.com/customize/)*. If the theme supports custom colors, you’ll see a “**Colors**” or “**Colors & Backgrounds** “section in the Customizer.
If that’s missing, the theme does not support custom colors. Consider switching to a [block theme,](https://wordpress.com/themes/filter/full-site-editing/) which gives you [full control over your site’s colors.](https://wordpress.com/support/custom-colors/)
## Access the Color Settings
To access the color settings in the [Customizer](https://wordpress.com/support/customizer/), follow the instructions below:
1. Visit your WordPress.com [dashboard](https://wordpress.com/home).
2. Navigate to *Appearance → [Customize](https://wordpress.com/customize)*.
3. Click on the color settings, which may be labeled **Colors** or **Colors & Backgrounds,** depending on your theme.
4. On the next screen, you will see a series of circles. Click one of the color circles to see which elements it controls. For example:
1. A circle with an “X” through it means the theme you are using did not set a color for that area. Learn about [your options for changing elements](https://wordpress.com/support/customizer/change-colors-in-the-customizer/#specific-elements) when it’s not built into the theme.
2. Click a color to change it. Several color suggestions for you to choose from will pop up. Click any of the suggested colors to use them for your site:
1. You’ll also see a link to **pick your own color** below that. Click that link to add a custom color [using the color picker.](https://wordpress.com/support/customizer/change-colors-in-the-customizer/#color-picker-in-the-customizer)
2. The **Default** button will clear your custom color selections and revert to the theme’s default colors.
3. Once you’re done making changes, click the **Save Changes** button at the top of the customizer to save your colors.
### Color Picker in the Customizer
You can choose a custom color using the color picker by clicking into each of the corresponding numbered sections, as explained below:
1. Click on the link labeled **pick your own color** to open up the color picker.
2. Click on the colored box section to pick a different shade of the current color.
3. Drag the rectangular icon up or down to select a different color.
4. If you already have a [HEX color code](http://htmlcolorcodes.com/) that you’d like to use, you can paste it into the box area. This code will automatically update if you choose a different color using the other options above.
The color picker in the Customizer has a safety net for accessibility. If you try to select a link color that is too similar to the background color, the color picker will automatically add contrast to keep your text readable.
## Background Color
For [classic themes](https://wordpress.com/support/theme-types/#classic-themes), you can select a background color or image if the theme supports it. Learn how to [change the background on your classic theme](https://wordpress.com/support/background-color/#tab-customizer).
## Specific Elements
If you do not see the option to update the color for a specific element on your site, the theme you are using does not have the option to manage that area through the customizer. You can either:
1. [Use Custom CSS](https://wordpress.com/support/editing-css/) to give that element a different color, or
2. Switch to a [block theme](https://wordpress.com/themes/filter/full-site-editing/) that gives you full control to customize the color of every element on your site.
