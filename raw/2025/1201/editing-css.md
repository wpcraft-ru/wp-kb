# Add custom CSS
Last reviewed on March 5, 2026
Copy page
While most changes can be made with settings in the [WordPress editor](https://wordpress.com/support/wordpress-editor/), you may prefer to use CSS to customize your site if you have experience writing CSS code. This guide will show you how to edit your website with CSS.
This feature is available on sites with the WordPress.com [Personal](https://wordpress.com/support/plan-features/personal-plan/), [Premium](https://wordpress.com/support/plan-features/premium-plan/), [Business](https://wordpress.com/support/plan-features/business-plan/), and [Commerce](https://wordpress.com/support/plan-features/ecommerce-plan/) plans. For free sites, [upgrade your plan](https://wordpress.com/support/manage-purchases/upgrade-your-plan/) to access this feature.

## About CSS
CSS stands for Cascading Style Sheets. It is a style sheet language that controls the appearance of HTML elements on a web page. On [eligible plans](https://wordpress.com/plans/), you can access a CSS editor to add your own CSS styles to override the default styles of your theme.
There is almost endless potential for what you can do with CSS to modify the design of a website. However, it requires knowledge of how CSS and HTML work (or at least a willingness to learn.) [The CSS guide from MDN web docs](https://developer.mozilla.org/en-US/docs/Web/CSS) is a great place to start learning CSS.
If you’re not interested in learning CSS, choose a [block theme](https://wordpress.com/support/theme-types/#block-themes-recommended) for your site. These themes provide the most flexible options to customize any aspect of your website’s design without any knowledge of computer code.
There are two methods to access your site’s CSS editor, depending on your theme. Each method is explained below.
## Access the CSS editor
To determine which section of this guide to follow, visit your [dashboard](https://wordpress.com/home) and check under *Appearance* on the left side. If you find *Editor* here, use the **Site Editor** section of this guide. Otherwise, use the **Customizer**instructions.
[Site Editor](https://wordpress.com/support/editing-css/#tab-site-editor)[Customizer](https://wordpress.com/support/editing-css/#tab-customizer)
You can add custom CSS to your site from the Styles panel in the editor by following these steps:
1. Visit your site’s [dashboard](https://wordpress.com/home).
2. Go to *Appearance → Editor*.
3. Select **Styles** from the sidebar.
4. Click the ellipses (**⋮**) and select “**Additional CSS**” option:
If you’re already editing in the Site Editor, you can also access it by clicking the Styles icon (it looks like a black and white circle) in the top toolbar, then clicking the ellipses (⋮) and selecting “**Additional CSS**“:
1. Type or paste your CSS into the text box provided.
2. Click the “**Save**” button at the top right of the screen to save the CSS to your site.
You can preview the impact of your CSS changes on any [block type](https://wordpress.com/support/wordpress-editor/blocks/) by loading the **Style Book**. Click the icon of an eye to toggle the Style Book preview on or off.
For themes that do not use the [Site Editor](https://wordpress.com/support/site-editor/), including [classic themes](https://wordpress.com/support/theme-types/#classic-themes) and many third-party themes, you can add CSS following these steps:
1. Visit your site’s dashboard.
2. Navigate to *Appearance → Customize → Additional CSS*.
3. Type or paste your CSS into the text box provided. The preview window on the right will reflect your changes.
4. Click the “**Save Changes**” button to save the CSS to your site.
**Media Width**
The “**Media Width**” option should be used if you have modified the width of the primary content area using custom CSS. The “Media Width” setting is the default size for full-size images when inserted into your site.
Note that it will not affect the size of some images you added before changing the setting, depending on how they were inserted, and you may have to re-insert some of them after changing the setting.
**Start Fresh**
By default, the custom CSS you add to the CSS editor will be loaded after the theme’s original CSS, which means that your rules can take precedence and override the theme’s styles.
You can turn the theme’s original CSS off entirely by clicking the “*Don’t use the theme’s original CSS”* checkbox. This will allow you to use any WordPress.com theme as a blank canvas for designing with CSS. This is an advanced option and should only be used if you want to start over and design the CSS for your theme from scratch.
If you want to build on top of the existing CSS rules—the most common and recommended approach—then you can leave this option disabled.
**Preprocessor**
WordPress.com has support for CSS preprocessors LESS and Sass (SCSS Syntax). This is an advanced option for users who wish to use CSS extensions like variables and mixins. See the [LESS](http://lesscss.org/) and [Sass](https://sass-lang.com/) websites for more information.
## Access CSS revision history
To determine which section of this guide to follow, visit your dashboard and check under *Appearance* on the left side. If you find *Editor* here, use the **Site Editor** section of this guide. Otherwise, use the **Customizer**instructions.
[Site Editor](https://wordpress.com/support/editing-css/#tab-site-editor)[Customizer](https://wordpress.com/support/editing-css/#tab-customizer)
Previous versions of your CSS are saved along with other style changes. To restore a CSS revision in the Site Editor, follow these steps:
1. Visit your site’s [dashboard](https://wordpress.com/home).
2. Go to *Appearance → Editor*.
3. Click the homepage preview on the right to open the editing screen.
4. Click the **“Styles”** icon (a circle with one white and one black half.) located in the top toolbar of the Site Editor.
5. Click the **“Revisions”** icon, which appears as a clock with an arrow pointing counterclockwise. The icon will only appear if there are saved revisions.
6. Click a revision to see how your site styles appeared in that version.
7. Click the **“Apply”** button when you locate the revision you want to use.
8. Click the **“Save”** button to save your changes.
If you would like to restore only the previous CSS but not any other styles in a revision, copy the CSS after applying a revision. Then, reload the Site Editor without saving anything, paste the copied CSS into Additional CSS, and save it.
The latest 25 versions of your CSS edits are saved and can be accessed by clicking “**See full history**” at the bottom of the CSS editor. You can restore previous versions of your CSS here.
If the option does not appear, there is no CSS history to restore.
## Add CSS to specific block types
In [block themes](https://wordpress.com/support/theme-types/), you can apply CSS code to specific blocks site-wide by following these steps:
1. Visit your site’s [dashboard](https://wordpress.com/home).
2. Navigate to *Appearance → Editor*.
3. Click on the **[Styles](https://wordpress.com/support/using-styles/)**menu option.
4. This time, select the **Blocks** section to access settings to customize the appearance of specific blocks for the whole site.
5. Click the name of the block you want to add CSS to. You can also locate your desired block type using the search box.
6. Scroll down to the bottom and click on **Advanced**.
7. In the box labeled “**Additional CSS**“, type CSS that will apply to all instances of that block type. When adding per-block CSS, you do not need to include a CSS selector—just add the property and value. An example is included in the video above.
8. Click the **Save** button at the top right of the screen to save the CSS to your site.
## Add CSS to one specific block
This section of the guide applies to plugin-enabled sites with our [WordPress.com Personal, Premium, Business, and Commerce plans](https://wordpress.com/pricing/). If your site has one of these plans, make sure to [activate it](https://wordpress.com/support/activate-your-plan/). For sites on the free plan, [upgrade your plan](https://wordpress.com/support/manage-purchases/upgrade-your-plan/) to access this feature.
You can add CSS directly to a single block on a single page. Unlike the methods above, this CSS applies only to that block on that page — it does not affect any other blocks of the same type elsewhere on your site.
1. Select the block you want to style.
2. Open the [block settings](https://wordpress.com/support/wordpress-editor/customize-blocks/#block-settings) in the sidebar.
3. Scroll down and expand the **Advanced** section.
4. In the “**Additional CSS**” field, enter your CSS. The CSS you enter here applies only to this block on this page.
1. **Save** or publish the page to apply the changes.
## CSS help
On eligible plans, you can add CSS to your site. WordPress.com does not offer direct support for writing or editing CSS. You can visit [About CSS](https://wordpress.com/support/editing-css/#about-css) to learn more about writing CSS, or you may wish to consider hiring a developer or designer. We recommend the WordPress experts available at the freelancer network called [Codeable](https://codeable.io/partners/wordpresscom/?ref=2ycmr).
If you contact us and request a change to your site that would require CSS, we may recommend:
* [Switching to a theme](https://wordpress.com/support/themes/changing-themes/) that allows you to achieve your goal without CSS.
* Hiring a dedicated developer or designer who you can work with to make in-depth changes to your website safely. See [Working With a Developer](https://wordpress.com/support/working-with-a-developer/) to learn more.
## Frequently asked questions
### Can I remove the WordPress.com footer credit with CSS?
The footer credit should not be modified with CSS. You can remove or modify the footer credit with settings depending on the type of theme the site uses:
* If your site uses a [block theme](https://wordpress.com/support/theme-types/), you can edit or remove the [footer credit](https://wordpress.com/support/footer-credits/) using the Site Editor—no CSS required.
* If your site uses a classic theme, you can [change the footer credit](https://wordpress.com/support/footer-credits/#tab-classic-themes) to a minimalist WordPress logo. You can hide the credit if the site has a [Business or Commerce plan subscription](https://wordpress.com/plans/).
### Can I use CSS rules like @import and @font-face?
Yes, on [plugin-enabled sites only.](https://wordpress.com/plans/)
### Can I use web fonts in CSS?
You can [choose your website fonts](https://wordpress.com/support/custom-fonts/) via settings. When working with CSS, you are limited to those two web fonts on the front end. However, you can [upload additional fonts](https://wordpress.com/support/upload-font/) on some plans.
### Can I upload images for use with my CSS?
Yes. You can upload an image to your [Media Library](https://wordpress.com/support/media/) and refer to it by its direct URL from within your CSS stylesheet. Here’s an example of how to use a background image in your stylesheet:
```
div#content {
background-image: url('http://example.files.wordpress.com/1999/09/example.jpg');
}
```
### Can I edit the CSS stylesheet directly?
We recommend making CSS edits using the CSS editor, as explained in this guide above. When you add CSS using this method, it overrides CSS rules from your theme’s stylesheet. This is a safe method that makes it easier to debug CSS conflicts and restore previous versions of CSS you’ve added.
If you are proficient in editing theme files directly, you can do so via [SFTP](https://wordpress.com/support/sftp/) or by [creating a child theme](https://wordpress.com/support/themes/child-themes/). We recommend testing changes on a [staging site](https://wordpress.com/support/how-to-create-a-staging-site/) before applying them to your live site.
### What happens if I cancel my WordPress.com plan?
All upgrades on WordPress.com are renewed yearly. Should you [cancel your subscription](https://wordpress.com/support/manage-purchases/cancel-a-purchase/), your custom CSS will still be saved, but it will no longer be applied to your site for others to see. If you would like them re-applied to your site, you can repurchase your plan, and the styles will be re-applied automatically, provided you haven’t changed themes. If you have changed themes, you will find your past CSS in the CSS revisions.
