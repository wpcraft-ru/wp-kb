# Get help with CSS
Last reviewed on January 14, 2026
Copy page
On eligible plans, you can alter your site’s design with [CSS code](https://wordpress.com/support/editing-css/). If you need assistance making CSS changes to your website, this guide will explain where you can find support.
This feature is available on sites with the WordPress.com [Personal](https://wordpress.com/support/plan-features/personal-plan/), [Premium](https://wordpress.com/support/plan-features/premium-plan/), [Business](https://wordpress.com/support/plan-features/business-plan/), and [Commerce](https://wordpress.com/support/plan-features/ecommerce-plan/) plans. For free sites, [upgrade your plan](https://wordpress.com/support/manage-purchases/upgrade-your-plan/) to access this feature.

## Look for alternatives to CSS
For those experienced with writing computer code, CSS can be a useful way to customize a theme. However, WordPress allows you to create a website without knowing any code, so knowledge of CSS is not needed to design a beautiful and professional site.
[Block themes on WordPress.com](https://wordpress.com/support/theme-types/#block-themes-recommended) provide the most flexible options to customize any aspect of your website’s design, thanks to the [Styles](https://wordpress.com/support/using-styles/) feature. If your site uses one of these themes, it’s unlikely you will need to add custom CSS, since you can fine-tune the display of each element in the block settings.
If you’re not sure if a change you’d like to make is possible without CSS, [get in touch with our Happiness Engineers](https://wordpress.com/support/help-support-options/) — we know the WordPress editor inside out and are happy to advise you on the best approach to take.
CSS is most commonly used to make design changes to [classic WordPress themes](https://wordpress.com/support/theme-types/#classic-themes), which have more limited customization options compared to block themes. If the changes you wish to make would require a lot of custom CSS that is beyond your ability to write, consider [changing themes](https://wordpress.com/support/themes/changing-themes/) now to make the process of updating your site’s design easier in the long term.
## CSS help
On eligible plans, you can add CSS to your site. WordPress.com does not offer direct support for writing or editing CSS. You can visit [About CSS](https://wordpress.com/support/editing-css/#about-css) to learn more about writing CSS, or you may wish to consider hiring a developer or designer. We recommend the WordPress experts available at the freelancer network called [Codeable](https://codeable.io/partners/wordpresscom/?ref=2ycmr).
If you contact us and request a change to your site that would require CSS, we may recommend:
* [Switching to a theme](https://wordpress.com/support/themes/changing-themes/) that allows you to achieve your goal without CSS.
* Hiring a dedicated developer or designer who you can work with to make in-depth changes to your website safely. See [Working With a Developer](https://wordpress.com/support/working-with-a-developer/) to learn more.
