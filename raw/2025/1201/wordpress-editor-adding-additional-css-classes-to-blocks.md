# Add additional CSS classes to blocks
Last reviewed on March 5, 2026
Copy page
If you are proficient in writing [CSS code](https://wordpress.com/support/editing-css/), you can add a CSS class to a block and style that block with CSS. This guide explains how to use the Additional CSS Class(es) feature of the [WordPress editor.](https://wordpress.com/support/wordpress-editor/)

## Define a CSS class on a block
To add a CSS class to a block, take the following steps:
1. Click on the block you are editing.
2. In the block settings on the right, click on the “**Advanced**” setting.
If you do not have the sidebar on the right, click the Settings icon in the top-right corner to bring up the settings. This icon looks like a square with two uneven columns:
1. In the box labeled “**Additional CSS Class(es)**“, define your class as either:
* A single word, such as `special`
* A term made up of multiple words with each word separated by hyphens, such as `special-class`
2. If adding multiple classes to the same block, separate each class by a space. For example: `special-class-1 special-class-2`
Once you have added the CSS class, it will reference the CSS you’ve defined [in the CSS Editor](https://wordpress.com/support/editing-css/#access-the-css-editor).
When defining your class, be careful not to use a class that may already have been defined in the website’s code. You’ll want to use unique terms not already in use within the site’s CSS.
📌
> You may know that when writing CSS code, we target a class by putting a period in front of the class, i.e. `.page`. However, do not place a period in front of the class when defining it in the block settings.
## Tips for using additional CSS classes
After you have defined a CSS class and added it to a block, you can write custom CSS code to target the class you created. Visit our [Add Custom CSS guide](https://wordpress.com/support/editing-css/) to learn where to add CSS to your website.
In the rest of this guide, we’ll share a few examples of how you can use Additional CSS Classes.
### Target specific blocks
Let’s say you want to add an animation to some [Image blocks](https://wordpress.com/support/wordpress-editor/blocks/image-block/) on your site, but leave all other images untouched. In this example, we define the `moving-image` class in the Advanced block settings:
Then, we can add the following CSS to our site to give this block a cool animation:
```
.moving-image {
animation-duration: 3s;
animation-name: slidein;
}
@keyframes slidein {
from {
margin-left: 100%;
width: 100%;
}
to {
margin-left: 0;
width: 100%;
}
}
```
This is the result when you open the page:
### Change the bullet icon in a list
You can change the icon used in a list from a circular disc to a square, an empty circle, a Roman numeral, and [many more!](https://css-tricks.com/almanac/properties/l/list-style/)
Define a class such as `square-bullet-list-block` like so:
Adding a custom class to the [List block](https://wordpress.com/support/wordpress-editor/blocks/list-block/)
Then, add this CSS to your [CSS editor](https://wordpress.com/support/editing-css/):
```
.square-bullet-list-block {
list-style-type: square;
}
```
and the result will be:
A list block with square bullets!
Depending on your theme’s design, you might need to add a class like- to the CSS above to target the correct class.
### Make use of existing custom CSS classes
Some classes come pre-defined in WordPress; you can add those to your blocks for cool effects.
For example, the class `has-drop-cap` will make the first letter of a paragraph block appear in a large font size:
The first letter is large because of the `has-drop-cap` class.
Removing the class will also remove the drop cap effect.
Below is a list of other existing CSS classes that some blocks will support, but remember that theme styles may override these existing classes. For best results, use one of the Recommended themes.
* `aligncenter`
* `alignright`
* `alignleft`
* `alignwide`
* `alignfull`
* `has-small-font-size`
* `has-large-font-size`
* `has-huge-font-size`
* `has-drop-cap`
## Add CSS to a specific block
Looking to add CSS code directly to a single block on a single page? You can use the **Additional CSS** field in the block’s Advanced settings. See [Add CSS to one specific block](https://wordpress.com/support/editing-css/#add-css-to-one-specific-block) for details.
## CSS help
On eligible plans, you can add CSS to your site. WordPress.com does not offer direct support for writing or editing CSS. You can visit [About CSS](https://wordpress.com/support/editing-css/#about-css) to learn more about writing CSS, or you may wish to consider hiring a developer or designer. We recommend the WordPress experts available at the freelancer network called [Codeable](https://codeable.io/partners/wordpresscom/?ref=2ycmr).
If you contact us and request a change to your site that would require CSS, we may recommend:
* [Switching to a theme](https://wordpress.com/support/themes/changing-themes/) that allows you to achieve your goal without CSS.
* Hiring a dedicated developer or designer who you can work with to make in-depth changes to your website safely. See [Working With a Developer](https://wordpress.com/support/working-with-a-developer/) to learn more.
