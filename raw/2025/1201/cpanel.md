# cPanel
Last reviewed on March 27, 2026
Copy page
Sites on WordPress.com do not use cPanel to manage common hosting features. WordPress.com uses a custom-designed control panel that includes all necessary features for web developers to manage their websites effectively. In this guide you will learn how to access the WordPress.com hosting features usually found in cPanel.

## WordPress.com hosting features
These are just some features available for your WordPress.com website. You can access various features via the [site’s settings](https://wordpress.com/hosting-config/).
### SSH
SSH, or Secure Shell, is a protocol that allows you to connect to services such as web servers. This lets you connect to our servers and manage your WordPress.com website via command-line tools such as [WP-CLI](https://developer.wordpress.com/docs/developer-tools/wp-cli/).
[Check out our complete guide on SSH.](https://wordpress.com/support/connect-to-ssh-on-wordpress-com/)
### SFTP
SFTP stands for Secure File Transfer Protocol (or SSH File Transfer Protocol) and is a way to access the files and folders on a website via a client program on your local computer.
[Learn more about using SFTP on WordPress.com.](https://wordpress.com/support/sftp/)
### phpMyAdmin
phpMyAdmin allows direct access to your site’s database. This is an advanced feature; accessing your database is not required for your site to function.
For more information, visit our [phpMyAdmin and MySQL guide](https://wordpress.com/support/phpmyadmin-and-mysql/).
### Where can I find server information?
Information regarding our server setup can be found in the [PHP environment](https://wordpress.com/support/php-environment/) support article.
