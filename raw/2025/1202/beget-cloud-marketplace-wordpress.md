**WordPress** – система управления содержимым сайта с открытым исходным кодом, написанная на PHP и выпущенная под лицензией GNU GPL версии 2. В качестве сервера базы данных используется MySQL. WordPress применяют для создания самых разных сайтов: личных блогов, сервисов для бизнеса, интернет-магазинов и т. д.
Эта CMS занимает первое место по популярности в мире. Она не требует знаний в сфере разработки: практически каждый может создавать и управлять файлами сайта через удобный веб-интерфейс.
**WordPress от Бегета** – это готовая сборка CMS со всем необходимым окружением. Дополнительных настроек не требуется. В Бегете вы можете установить WordPress на VPS и начать пользоваться системой буквально в несколько кликов.

## Инструкция по установке WordPress
При создании сервера, помимо стандартных параметров, вам будет предложено указать:
* Домен, на котором вы будете использовать сервер для WordPress. Он будет использоваться для доступа к панели управления CMS.
Мы автоматически установим на него SSL-сертификат бесплатно.
* Название вашего ресурса.
* Логин администратора. Он будет использоваться для входа в ПУ WordPress с правами администратора.
* E-mail администратора. Он будет указан в WordPress в качестве контактного и в качестве e-mail администратора при выпуске SSL-сертификата.
* Пароль администратора.
* Рекомендуется установить дополнительные плагины (опционально):
+ Elementor – плагин, позволяющий превратить WordPress в настоящий конструктор сайтов;
+ YoastSEO – плагин, помогающий продвижению сайта в поисковой выдаче.
Установив и настроив данные плагины, вы сможете оптимизировать внешний интерфейс и работу вашего сайта. Готово, вы успешно установили WordPress!
## Использование WordPress
Для доступа в ПУ WordPress вам необходимо перейти по ссылке `https://my_domain.com/wp-admin`.
Вам понадобится ввести данные учетной записи администратора, которые вы указывали, создавая виртуальный сервер для WordPress. Мы также отправим их вам на почту, которую вы указали при создании сервера.
Удачной работы с вашим новым сайтом на WordPress!
## FAQ
**Где находятся файлы сайта на моем WordPress-сервере?**
Файлы вашего сайта находятся в директории `/var/www/wordpress`
**Где находится конфигурация Nginx на моем WordPress-сервере?**
Основной конфигурационный файл вашего веб-сервера Nginx: `/etc/nginx/nginx.conf`
Конфигурационный файл вашего виртуального хоста: `/etc/nginx/sites-enabled/wordpress`
**Где находится конфигурация Apache на моем WordPress-сервере?**
Конфигурационный файл вашего виртуального хоста: `/etc/apache2/sites-enabled/wordpress.conf`
**Где находится конфигурация PHP на моем WordPress-сервере?**
Основной конфигурационный файл: `/etc/php/8.4/apache2/php.ini`
Дополнительные файлы конфигурации в директории: `/etc/php/8.4/apache2/conf.d`
**Как изменить домен для сайта?**
* Перейдите в ПУ WordPress: `https://my_domain.com/wp-admin`
* Через меню слева перейдите в раздел "Настройки".
* Отредактируйте адрес вашего ресурса в полях "Адрес WordPress (URL)" и "Адрес сайта (URL)".
* Сохраните изменения внизу страницы.
Обратите внимание!
Перед сменой домена необходимо поменять A-запись домена, который вы собираетесь использовать для сайта. В качестве значения для А-записи указывается IP-адрес вашего виртуального сервера (VPS).
