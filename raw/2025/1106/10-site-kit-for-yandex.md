## Общая информация

- **Тип:** Панель интеграции сервисов Яндекса (не классический SEO-плагин)
- **Аналог:** Site Kit by Google (официальный плагин от Google)
- **Позиционирование:** «Единая панель управления сервисами Яндекса для WordPress»
- **Целевая аудитория:** Владельцы сайтов на WordPress в РФ
- **Лицензия:** Бесплатный

---

## Интегрируемые сервисы Яндекса

| Сервис | Назначение | Статус |
|--------|-----------|--------|
| **Яндекс Метрика** | Веб-аналитика, посещаемость, поведение пользователей | ✅ |
| **Яндекс Вебмастер** | Контроль индексации, позиции в поиске, технические ошибки | ✅ |
| **IndexNow** | Мгновенное уведомление о новых/обновлённых страницах | ✅ |
| **Яндекс ID** | OAuth-авторизация через учётную запись Яндекса | ✅ |
| **Яндекс РСЯ** | Размещение рекламных блоков | Заявлено |
| **Yandex Object Storage** | Облачное хранилище для масштабирования | Заявлено |

---

## Ключевые возможности

### Dashboard и мониторинг
- **Yandex Overview** — сводная страница с показателями сайта
- **Yandex URL Inspector** — проверка статуса индексации конкретных URL
- Данные Яндекс Метрики прямо в админке WordPress
- Отслеживание трафика, источников переходов, поведения пользователей

### Быстрая индексация (IndexNow)
- Автоматическое уведомление Яндекса об изменениях на сайте
- Новые статьи и обновления попадают в поиск быстрее
- Не нужно вручную отправлять страницы на переиндексацию

### Безопасность
- Авторизация через Яндекс ID OAuth
- Защита данных через официальный протокол Яндекса

## Преимущества

- **Всё в одном месте** — не нужно открывать несколько вкладок и сервисов
- **Простая настройка** — пошаговая авторизация через OAuth за минуты
- **Официальные API** — данные от Яндекса, а не от сторонних источников
- **Бесплатно** — полностью бесплатный плагин
- **Адаптирован под РФ** — ориентирован на российскую поисковую экосистему

## Отличия от классических SEO-плагинов

Site Kit for Yandex **не заменяет** SEO-плагины типа Rank Math или Yoast. Это другой класс инструментов:

| Функция | Site Kit for Yandex | SEO-плагины (Rank Math и др.) |
|---------|---------------------|-------------------------------|
| Мета-теги (title/description) | ❌ | ✅ |
| Schema-разметка | ❌ | ✅ |
| XML Sitemap | ❌ | ✅ |
| Редиректы | ❌ | ✅ |
| Анализ читаемости | ❌ | ✅ |
| Яндекс Метрика | ✅ | Частично |
| Яндекс Вебмастер | ✅ (панель) | ✅ (верификация) |
| IndexNow | ✅ | ✅ (Rank Math, SEOPress) |
| URL Inspector | ✅ | ❌ |
| Реклама РСЯ | ✅ | ❌ |

**Вывод:** Site Kit for Yandex — комплементарный инструмент. Он дополняет SEO-плагин данными из экосистемы Яндекса, а не заменяет его.

## Сравнение с Site Kit by Google

| Функция | Site Kit by Google | Site Kit for Yandex |
|---------|-------------------|---------------------|
| Веб-аналитика | Google Analytics 4 | Яндекс Метрика |
| Инструменты вебмастера | Search Console | Яндекс Вебмастер |
| Реклама | Google AdSense | Яндекс РСЯ (заявлено) |
| Скорость страниц | PageSpeed Insights | ❌ |
| Управление тегами | Google Tag Manager | ❌ |
| A/B-тесты | Google Optimize | ❌ |
| Быстрая индексация | ❌ | IndexNow |
| Инспектор URL | ❌ | Yandex URL Inspector |
| Авторизация | Google OAuth | Яндекс ID OAuth |
| Данные на уровне поста | ✅ | ❌ |
| Зрелость | 2019, 3+ млн установок | Молодой проект |
| Поддержка | Официальная Google | Один разработчик |

---

## Чего не хватает

- **Нет аналога PageSpeed Insights** — производительность проверяется отдельно
- **Нет Tag Manager** — события и теги вручную
- **Нет A/B-тестирования** — Яндекс Эксперименты не интегрированы
- **Молодой проект** — один разработчик, нет гарантий долгосрочной поддержки
- **Нет данных на уровне поста** — только общая картина по сайту

---

## Рекомендация

**Идеальный стек для сайта на WordPress с ориентацией на РФ:**

1. **SEO-плагин** (Rank Math / SEOPress) — мета-теги, Schema, sitemap, редиректы
2. **Site Kit for Yandex** — Метрика, Вебмастер, IndexNow в админке
3. **WP Rocket** (или аналог) — производительность и кеширование

SEO-плагин и Site Kit for Yandex не конфликтуют, решают разные задачи. Можно использовать вместе.
