# Миграция между SEO-плагинами WordPress

> Дата сбора: 2026-05-03
> Источники: rankmath.com, easyseo.pro, seopress.org, web_search

---

## Главное правило

> **Миграция SEO-плагинов НЕ теряет SEO-данные.** Все современные плагины умеют импортировать метаданные (title, description) из конкурентов.

---

## Подготовка к миграции

### Обязательно
1. **Создать полный бэкап сайта** (критично!)
2. Задокументировать текущие настройки:
   - Топ-ключевые слова
   - Ручные редиректы
   - Особые настройки Schema
3. Установить новый плагин **при активном старом** (только на время импорта)

### Важное предупреждение
> **Не держите два SEO-плагина активными одновременно после миграции.** Это создаёт дублирование метаданных, конфликтующую схему и может сломать настройки.

---

## Путь #1: Yoast → Rank Math

### Пошагово

1. **Deactivate Yoast SEO** (деактивировать)
2. **Install and Activate Rank Math**
3. Запустить **Setup Wizard**
4. Выбрать **"Import from Yoast SEO"**
5. Нажать **Start Import** — прогресс виден в status bar
6. Опционально: пересчитать SEO-оценки для всех страниц

### Особенности
- Импортируются: title, description, ключевые слова, настройки Schema
- Может потребоваться ручная очистка данных Yoast из БД
- Проверить настройки sitemap после импорта

### Команда для очистки Yoast после миграции
- WordPress Dashboard → Tools
- Yoast SEO tab → "Delete Data"

---

## Путь #2: Yoast → SEOPress

### Пошагово

1. Установить SEOPress при активном Yoast
2. SEOPress → **Import/Export**
3. Выбрать **"Import from Yoast SEO"**
4. Выбрать типы данных для импорта:
   - Мета-заголовки и описания
   - Настройки соцсетей
   - Редиректы
   - Schema
5. Запустить импорт
6. Деактивировать Yoast

### Особенности
- SEOPress импортирует метаданные для постов, страниц, CPT и таксономий
- Поддерживается импорт из CSV (для сложных случаев)

---

## Путь #3: Другие миграции

### AIOSEO → Rank Math
- Setup Wizard → Import from AIOSEO

### Любой плагин → Rank Math
- Advanced Settings → Import & Export
- Поддержка: Yoast, AIOSEO, SEOPress, Rank Math (старая версия)

### Любой плагин → SEOPress
- SEO → Tools → Import/Export
- Поддержка: Yoast, Rank Math, AIOSEO, WP Meta SEO

---

## Проверка после миграции

### Контрольный список

- [ ] Главная страница: title и description корректны
- [ ] Тестовый пост: метаданные на месте
- [ ] Страница товара (WooCommerce): Product Schema
- [ ] XML Sitemap доступен и содержит все страницы
- [ ] Robots.txt корреректен
- [ ] Нет битых ссылок (редиректы работают)
- [ ] Schema-разметка валидна (Google Rich Results Test)
- [ ] Социальные превью работают (OG, Twitter)
- [ ] Google Search Console / Яндекс.Вебмастер: sitemap принят

---

## Возможные проблемы и решения

| Проблема | Решение |
|----------|---------|
| Частичный импорт | Использовать "Import & Export" вручную, реимпортировать |
| Дубли описаний | Проверить критичные страницы вручную, поправить |
| Потеря редиректов | Экспортировать редиректы из старого плагина до деактивации |
| Сломанная Schema | Перегенерировать Schema, проверить валидатором |
| Остатки данных Yoast | Tools → Delete Data (Yoast) |

---

## Мониторинг после миграции

- **Первая неделя:** ежедневный мониторинг позиций
- **Первый месяц:** еженедельная проверка GSC / Яндекс.Вебмастер
- Правильно выполненная миграция **не вызывает падения позиций**

---

## Рекомендация по выбору плагина для миграции

Если мигрируете с Yoast:

1. **Rank Math** — если нужен максимум функций бесплатно
2. **SEOPress** — если нужен чистый UI, white-label, Яндекс

Оба поддерживают полный импорт из Yoast.
