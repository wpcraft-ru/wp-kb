## Ключевые возможности

### Бесплатная версия

**Контент и анализ**
- **Неограниченное количество ключевых слов** для анализа
- Анализ контента в реальном времени
- Предпросмотр сниппетов Google
- Социальные превью (Open Graph, Twitter Cards)

**Техническое SEO**
- Автоматическая XML и HTML карта сайта
- Карты изображений
- Управление robots.txt
- **Редиректы** (бесплатно!)
- **404 мониторинг** (бесплатно!)

**Schema**
- Встроенная Schema-разметка
- Типы: Article, Product, FAQ, Event, Local Business

**Интеграции**
- Google Analytics
- Google Search Console
- Microsoft Clarity
- **Яндекс.Вебмастер** (верификация, синхронизация данных)

---

### PRO-возможности ($49/год)

**AI-генерация мета-тегов**
- Интеграция с OpenAI (ChatGPT), Google Gemini, Claude, MistralAI, DeepSeek
- Автоматическая генерация meta title и description
- Генерация Open Graph и Twitter Card тегов
- Генерация alt-текстов для изображений
- Массовые операции для крупных сайтов
- Настройка промптов, языков, API-ключей

**Продвинутое SEO**
- Карты видео и новостей
- Аудит сайта
- Расширенные Schema-типы
- Автоматические внутренние ссылки

## Плюсы и минусы

### ✅ Плюсы
- Бесплатная версия очень функциональна
- ИИ-генерация мета-тегов (5 провайдеров: OpenAI, Claude, Gemini, Mistral, DeepSeek)
- Чистый интерфейс без рекламы
- White-label для агентств
- Выгодная цена (Unlimited $149/год)
- Интеграция с Яндекс.Вебмастер
- Лёгкий перенос с других SEO-плагинов

### ❌ Минусы
- Меньше узнаваемости, чем Yoast/Rank Math
- Меньше документации и туториалов
- Интерфейс менее «обучающий» для новичков
- AI-функции только в PRO

## Для кого подходит

- Агентства и фрилансеры (white-label, unlimited)
- Разработчики, ценящие чистоту и скорость
- Те, кому нужна ИИ-генерация мета-тегов
- Пользователи из РФ (Яндекс-интеграция)
