## Rank Math + Яндекс

### Верификация
1. **Rank Math SEO → General Settings → Webmaster Tools**
2. Вставить Яндекс Verification ID
3. Rank Math автоматически добавляет мета-тег на сайт

### Sitemap
- Ручная отправка через **Indexing → Sitemap files**
- Автоматическая генерация XML-карты

### IndexNow
- Поддержка Yandex IndexNow для мгновенной индексации
- Уникальная функция — не у всех конкурентов

### Примечание
- Некоторые функции требуют **Advanced Mode**

---

## SEOPress + Яндекс

### Верификация
- Вставка мета-значения Яндекса в настройках SEOPress
- SEOPress → Search Engine Settings → Yandex

### Sitemap
- Автоматическая отправка sitemap через настройки плагина
- Поддержка XML, HTML, изображений, видео, новостей

### IndexNow
- Поддерживается

### Синхронизация данных
- SEOPress синхронизирует данные вебмастера аналогично Google Search Console
- Быстрый доступ к информации прямо из админки

---

## Yoast SEO + Яндекс

### Верификация
- **Yoast SEO → Settings → Site connections**
- Вставить код верификации Яндекса

### Примечания
- Только верификация — нет продвинутой интеграции
- Sitemap отправляется отдельно через Яндекс.Вебмастер

---

## Рекомендация

Для сайтов, ориентированных на русскоязычную аудиторию:

1. **Rank Math** — лучший выбор: верификация, sitemap, IndexNow
2. **SEOPress** — хорошая альтернатива с аналогичной функциональностью
3. **Yoast** — только базовая верификация
