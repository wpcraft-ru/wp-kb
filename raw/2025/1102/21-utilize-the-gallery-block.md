# Utilize the Gallery block

**Source:** https://learn.wordpress.org/lesson/utilize-the-gallery-block/
**Course:** Beginner WordPress Designer
**Section:** Working with media blocks

---

## Transcript

### Introduction
Gallery block — multiple photos automatically arranged in a gallery. Perfect for portfolios, product images, photo collections.

### Adding and using
- **Drag and drop** images from computer → auto-creates Gallery block
- Or type `/gallery` → upload from Media Library
- Always add **alt text** (screen readers + SEO)

**Settings:**
- Number of columns
- Image resolution (thumbnail, medium, large, full)
- **Crop images to fit** (on by default) — makes images in each row same size

### Styles
- Background color
- Block spacing between images (horizontal + vertical)
- Padding and margin (enable via three dots)
- Border and radius

**Example:** 5 columns → vertical block spacing 0 → full width alignment → images united.

### Responsiveness
Gallery block is responsive by default.

### Lightbox
Enable globally: Site Editor → Styles → Style Book → Media → Image block → Settings → **"Expand on click"**. Images open in overlay on same page.

### Customizing patterns
- Change columns → bulk select all images → change radius/border in one go
- Background color + padding + white 10px border on all images

### Additional options
- Add link and caption to Gallery
- Reposition images via arrows or drag-and-drop
- Replace individual images
- Add more images via "Add" in toolbar
- Rename block or create pattern (three dots)

## Practical
1. Enable lightbox in Style Book
2. Download 5 elephant images from Unsplash
3. **Example 1:** 3 columns, no special alignment
4. **Example 2:** 1 column, wide width, block spacing 0
