# Beginner WordPress Designer
This learning pathway equips designers new to WordPress with the skills to create professional WordPress websites. You’ll learn to build a style guide, understand block themes, and design with a block-first approach. Explore best practices in web design, customize themes, master layout composition, and work with media blocks.
Some lessons in this learning pathway are duplicates of those in the user learning pathway.
