# Designing with the Columns block

**Source:** https://learn.wordpress.org/lesson/designing-with-the-columns-block-copy/
**Course:** Beginner WordPress Designer
**Section:** Layout composition

---

## Transcript

### Introduction
The Columns block is a container block for custom layouts — up to **6 columns**. Used one after another, creates a grid effect.

### Adding and customizing
Insert via Inserter or `/columns`. Choose number and size of columns. Add blocks inside each column: Images, Headings, Paragraphs, Cover blocks, etc.

### Block Toolbar
- **Parent-child relationship** — main Columns block + individual columns
- Alignment: wide-width, full-width
- **Duplicate column:** three dots → Duplicate (saves time)
- **Move column:** arrows in toolbar or drag-and-drop in List View

### Sidebar Settings
- **Add column:** click Inserter between columns or use parent block settings
- **Stack on Mobile** — toggle to stack columns vertically on mobile
- **Styles:** colors (parent + individual columns), typography (font, appearance, letter spacing), size
- **Dimensions:** padding, margin, space between columns
- **Border + radius**, drop shadow

### Transform
Other blocks can be transformed into Columns block, and Columns into Group block.

## Practical
Re-create 3-column design:
1. Add Columns block with 3 equal columns
2. **Left column:** 5px border, padding, Paragraph "01" (75px, bold, line-height 1.3, center), "Consultation" (Large, line-height 4, center), Lorem ipsum text (center)
3. **Duplicate** first column twice, remove empty columns, update content
