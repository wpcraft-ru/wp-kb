## Transcript

### Introduction
Media and Text block — combines images/videos with text side by side. Ideal for: feature highlights, about me, meet the team, contact us sections.

### Adding media and text
Type `/media and text` → add image (with alt text) → add content on the right.

**Example workflow:**
1. Add image from Media Library
2. Add text → transform first paragraph to Heading (XL)
3. Add Buttons block (outline style) as CTA

### Customization
- **Alignment:** wide width, full width
- **Image position:** left or right (swap in toolbar)
- **Background color:** custom
- **Padding:** space around content
- **Media width:** slider to adjust image-to-text ratio
- **Stack on mobile** (default on) — content stacks vertically on small screens
- **Crop image to fill** — when image doesn't fit column dimensions. Use **focal point picker** to center important part after cropping
- **Vertical alignment:** top, middle, bottom
- **Border and radius** available

### Using patterns
Inserter → "Media and text" → ready-to-use patterns from theme. Use as base or inspiration. Customize: change background, replace image → pattern adapted in few clicks.
