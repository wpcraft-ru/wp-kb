## Claude Code для маркетолога

Claude Code (CLI-инструмент Anthropic) — не только для разработчиков. Маркетологи используют его для задач, которые раньше требовали команды разработки:

- Авто-обновление контента
- Внутренняя перелинковка
- Конкурентный анализ
- Генерация маркетинговых материалов
- AEO-мониторинг

## Что такое Skills

Skills в Claude — переиспользуемые инструкции в формате Markdown. Claude загружает skill в контекст при срабатывании триггера и выполняет задачу как по плейбуку. Это главная точка входа для маркетологов:

```
Claude, используя skill "homepage-review", проанализируй homepage competitor.com
```

Claude загружает файл `homepage-review.md` с критериями оценки и выдаёт структурированный результат.

## Кейсы маркетологов

### 1. Проверка позиционирования (Emily Kramer, MKT1)

Skill оценивает B2B-лендинг по фреймворку позиционирования:
- Проверяет hero-секцию и всю страницу отдельно
- Выставляет letter grade (A/B/C/D)
- Даёт конкретные исправления, включая переписанные заголовки
- Построен за 4 итерации тестирования на реальных страницах

### 2. Customer Lookalike Outbound (Elaine Zelby, Tofu)

Еженедельный агент:
- Смотрит closed-won сделки в HubSpot
- Находит 10 похожих компаний
- Подбирает 3-5 контактов на компанию через Clay
- Генерирует 4-email sequence + LinkedIn DM
- Отправляет черновики в Slack на проверку команде

### 3. Авто-обновление контента (Firecrawl)

- Ahrefs находит «увядающие» посты (падающий трафик)
- Уведомление в Slack
- Claude обновляет: статистику, примеры кода, секции, внутренние ссылки
- Human-in-the-loop: проверка перед публикацией

### 4. Агент внутренней перелинковки (Firecrawl)

- Claude имеет доступ ко всему репозиторию сайта
- Семантически подбирает релевантные внутренние ссылки
- Знает приоритетные страницы для продвижения
- Заменил SaaS-инструмент за $30/мес

### 5. Конкурентный анализ

```
Use Firecrawl to scrape the pricing pages of these 5 competitors
and make a comparison table.
```

Claude + Firecrawl = автоматический сбор и структурирование данных конкурентов.

## Технические маркетинговые задачи

- **Оптимизация изображений:** WebP-конвертация (30-40% меньше размера)
- **Аудит скорости:** Lazy loading, JS-бандлы, third-party скрипты
- **FAQ-генерация:** через Firecrawl Agent → structured data для AEO
- **Файловые конверсии:** CSV → JSON, реструктуризация данных

## AEO-мониторинг через AI

Claude анализирует crawl-данные (Vercel bot) и определяет, какие вопросы пользователи задают сайту. Это напрямую питает AEO-стратегию (Answer Engine Optimization):

- Какие вопросы задают AI-поисковики (ChatGPT, Perplexity)?
- Какие ответы получают?
- Где контент нужно улучшить для попадания в AI-ответы?

## ИИ и SEO/GEO (российский контекст)

По данным DP.ru (2025):

- Органический трафик снизился на 15-40% из-за AI-ответов в поиске
- 30-50% запросов будут закрываться AI-ответами в ближайшие 1-2 года
- Симбиоз SEO и GEO (продвижение для AI-ответов) становится решающим

**Как попасть в AI-ответы:**
- Упоминания в крупных отраслевых изданиях
- Регулярное обновление данных (цены, услуги) на всех площадках
- Структурирование информации: списки, таблицы, вопрос-ответ
- Регулярная проверка отображения компании в AI-ответах

## Как начать маркетологу

1. **Выучите основы технического стека** — HTML, CSS, как работает JS, SEO-фундамент
2. **Автоматизируйте рутину в первую очередь** — еженедельная занятость, отнимающая часы
3. **Создайте skills из повторяемых процессов** — code review → copy review, strategy review
4. **Начинайте с фазы 1, валидируйте, затем фаза 2** — не пытайтесь автоматизировать всё сразу
5. **Получите review инженера** — для customer-facing изменений
6. **Human-in-the-loop лучше полной автономии** — проверяйте результат AI перед публикацией
