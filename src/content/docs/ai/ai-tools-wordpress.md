## AI и WordPress

Искусственный интеллект всё активнее используется в экосистеме WordPress — от генерации контента до создания целых сайтов. Возможности различаются между WordPress.com и self-hosted. Подробнее об архитектуре AI в WordPress читайте в [AI-архитектуре WordPress](./wp-ai-architecture).

## ⚠️ AI Website Builder (только WordPress.com)

WordPress.com предлагает встроенный AI Website Builder:
- Создание полностью готового сайта по текстовому описанию
- Кастомизация через чат с AI в сайдбаре
- Использование темы Assembler
- Бесплатно для создания; для запуска нужен платный план

## Альтернативы для self-hosted WordPress

### AI-контент и генерация сайтов

- **AI Engine** (Jordy Meow) — ChatGPT-интеграция: генерация контента, изображений, чат-боты, отслеживание токенов
- **Bertha AI** — AI-ассистент в Gutenberg, Elementor и Divi: текст, изображения, многоязычный
- **10Web AI Builder** — описываете сайт → AI генерирует готовый сайт на Elementor; сокращает время разработки до 70%
- **AI Power** — генератор контента, AI-формы, чат-боты
- **Jetpack AI Assistant** — генерация и улучшение текстов прямо в редакторе

### Генерация изображений

- **Stable Diffusion** — через плагины или сторонние API
- **Unsplash** — бесплатный сток в редакторе WordPress (встроен)
- **Pexels / Pixabay** — через плагины для поиска royalty-free изображений

### Внешние AI-инструменты для дизайна

- **ChatGPT / Claude** — генерация структуры сайта, текстов, идей для дизайна
- **Midjourney / DALL-E** — генерация уникальных изображений
- **Figma AI** — AI-помощь в дизайне макетов

## Советы по промптам для AI

- **Будьте конкретны:** «сделай кнопку синей» лучше чем «измени кнопку»
- **Указывайте секцию:** «измени заголовок в блоке Услуги»
- **Простым языком:** «сделай текст короче», «используй яркие цвета»
- **Запрашивайте идеи:** «какие изображения подойдут для этой страницы?»
- **Разбивайте на шаги:** обновляйте по одной секции за раз

## Ограничения AI

- AI-контент требует проверки и редактуры человеком
- Сгенерированные изображения могут иметь артефакты
- AI не понимает юридических и регуляторных требований
- E-commerce функциональность обычно требует ручной настройки
- При чрезмерной опоре на AI дизайны могут становиться шаблонными
