## Обзор

Сервисы этой категории решают задачи конверсии трафика в лиды и продажи:
1. **Лидогенерация** — попапы, чаты, обратный звонок, формы захвата
2. **CRO (Conversion Rate Optimization)** — A/B-тестирование, персонализация, heatmaps
3. **Коммуникация с посетителем** — онлайн-чат, омниканальные диалоги

## Российские сервисы

### Envybox

Мульти-сервис лидогенерации: чат, обратный звонок, «генератор клиентов», мультикнопка. Поведенческие триггеры (всплывающие окна на выходе, автоприглашения в чат, таймеры).

- **Коллтрекинг:** источник звонка → кампания → ключевое слово, запись разговоров
- **CRM-модуль:** ФИО, телефон, город, источник, поисковый запрос
- **Интеграция:** через ApiMonster или прямой JS-код. Связка с WooCommerce, Contact Form 7, Roistat
- **Минусы:** заточен под русскоязычный рынок
- **Для кого:** интернет-магазины, услуги, B2B/B2C в РФ

### CallbackHunter

Виджет «перезвоните мне»: посетитель оставляет телефон → за 20–30 секунд соединяет с менеджером. Ring-group с автодозвоном по цепочке.

- **Триггеры:** on exit, по времени, страницам, кликам. Настройка часов работы.
- **Интеграция:** JS-код в `footer.php` или Custom HTML виджет. Сторонние плагины-обёртки на GitHub.
- **Минусы:** нет полноценного официального плагина в репозитории WordPress.org
- **Для кого:** сайты, где важен быстрый захват звонков

### Jivo (JivoSite)

Омниканальный чат — официальный плагин **JivoChat Live Chat** для WordPress (2–3 минуты на установку).

- **Каналы:** чат на сайте + WhatsApp + Telegram + VK + телефон + email — все диалоги в одном интерфейсе
- **Чат-боты:** авто-сбор лидов (email, телефон, город), триггеры и pop-up формы
- **Плюсы:** омниканальность, базовая аналитика по лидам, бесплатный тариф
- **Минусы:** для глубокой автоматизации (WooCommerce + CRM + email) нужны доп. инструменты
- **Для кого:** все типы сайтов на WordPress — от лендингов до магазинов

## Международные сервисы

### OptinMonster

Лидер в лидогенерации: popups, slide-ins, floating bars, countdown timers, full-screen campaigns. **Exit-intent технология** улавливает уходящих посетителей.

- **700+ шаблонов**, drag-and-drop builder
- **Таргетинг:** страницы, категории, поведение, гео, referral, scroll depth, cookies
- **Интеграции:** Mailchimp, Brevo, ActiveCampaign, HubSpot, Salesforce и десятки других
- **A/B тестирование** и campaign playbooks
- **Минусы:** дорого для маленьких сайтов, learning curve
- **Цены:** Basic $9/мес → Plus $19/мес → Pro $29/мес → Growth $49/мес
- **Для кого:** устоявшиеся WordPress-сайты с хорошим трафиком

### VWO (Visual Website Optimizer)

Enterprise CRO платформа: A/B, multivariate и split-URL тесты с визуальным редактором (менять текст, кнопки, layout без кода).

- **VWO Insights:** heatmaps, session recordings, form analytics, surveys
- **AI-фичи:** генерация идей для тестов, сегмент-предложения, ROI-приоритизация
- **Интеграция:** лёгкий WordPress-плагин или JavaScript snippet
- **Минусы:** overkill для простых проектов, enterprise-цены
- **Для кого:** mid-market / enterprise команды с серьёзной CRO-программой

### Convert Experiences

Гибкое A/B-тестирование и персонализация. 40+ фильтров таргетинга, Frequentist и Bayesian статистика.

- **Плагин Convert Experiments:** авто-вставка сниппета, передача метаданных (page type, category, tags)
- **Плюсы:** гибкое тестирование, сильный таргетинг, оба stats-движка
- **Минусы:** плагин минимален (управление в облаке), цена высоковата для простых тестов
- **Для кого:** SMB и продуктовые команды, которым нужен Optimizely-подобный функционал без enterprise-цен

## Как выбрать

1. **Омниканальный чат (бесплатный старт)** → Jivo
2. **Захват уходящих посетителей** → OptinMonster
3. **Быстрый обратный звонок** → CallbackHunter
4. **Мульти-виджеты + коллтрекинг** → Envybox
5. **A/B-тестирование для SMB** → Convert Experiences
6. **Enterprise CRO** → VWO
