## Обзор

Аналитика для WordPress работает на трёх уровнях:
1. **Счётчики и сбор данных** — сколько посетителей, откуда, какие страницы
2. **Поведенческий анализ** — heatmaps, записи сессий, воронки
3. **Сквозная аналитика** — связка «реклама → сайт → лид → продажа → ROI»

Стандартная рекомендация: один счётчик (GA4 или Яндекс.Метрика) + один поведенческий инструмент (Clarity или Hotjar) + сквозная аналитика для e-commerce.

## Счётчики и базовая аналитика

### Яндекс.Метрика

Полноценная бесплатная замена Google Analytics для РФ/СНГ. Официальный плагин для WordPress: авто-установка счётчика, e-commerce отслеживание без кода, дашборд-виджет в админке.

- **Особенности:** Вебвизор (session replay), click-карты, отчёты по часам/дням/месяцам
- **Новое в 2025:** офлайн-конверсии, конверсии из мессенджеров (WhatsApp, Telegram, Viber)
- **Для кого:** блоги, магазины, корпоративные сайты с аудиторией РФ

### Google Analytics 4 (GA4)

Глобальный стандарт веб-аналитики. Event-based tracking вместо pageviews. Интегрируется через множество WordPress-плагинов:

| Плагин | Особенности |
|--------|-------------|
| **MonsterInsights** | Полный eCommerce, дашборды в WP, GDPR |
| **GA4WP** | Лёгкий дашборд, IP-анонимизация |
| **HT Easy GA4** | WooCommerce + Google Ads events |
| **Site Kit by Google** | GA4 + Search Console + AdSense |
| **AnalyticsWP** | GDPR-фокус, per-page stats |

Все плагины обеспечивают: авто-вставку gtag.js, авто-трекинг ссылок и скачиваний, scroll depth, WooCommerce enhanced eCommerce.

## Поведенческий анализ

### Microsoft Clarity

**Полностью бесплатный** инструмент от Microsoft без ограничений по трафику. Session recordings с AI-суммаризацией (Clarity Copilot), heatmaps, scroll-depth, rage-click и dead-click detection.

- **Плюсы:** бесплатно без ограничений, AI-суммаризация (250 записей → 1 отчёт), лёгкий скрипт, авто-маскировка чувствительных данных
- **Минусы:** нет surveys и feedback widgets (в отличие от Hotjar)
- **Идеальная связка:** GA4 = «что происходит», Clarity = «почему»
- **Важно:** с октября 2025 требует cookie consent для EEA/UK/Switzerland

### Hotjar

Heatmaps + session recordings + on-page surveys + feedback widgets — всё в одном. AI-тегирование записей (rage clicks, form abandonment).

- **Плюсы:** наглядные визуальные отчёты, AI-ассистент для поиска UX-проблем
- **Минусы:** бесплатный тариф маловат (300 записей), платные быстро дорожают, может замедлять сайт
- **Для кого:** сайты с бюджетом на аналитику, нужны опросы и feedback

## Сквозная аналитика и коллтрекинг

### Roistat

Связывает полную цепочку: посетитель сайта → заявка → звонок → сделка → прибыль. Интеграции с WooCommerce, Contact Form 7, Gravity Forms.

- **Функции:** AI-прогноз продаж, модели атрибуции, авто-отключение убыточных рекламных кампаний
- **Плюсы:** единый дашборд (сайт + CRM + 1С + реклама), ROI по каждому каналу
- **Минусы:** высокий порог входа по цене, сложная настройка, российский фокус
- **Для кого:** интернет-магазины с рекламными бюджетами

### Calltouch

Коллтрекинг с динамической подменой номеров под канал трафика. Связывает звонки с визитами и формами. Официальный WordPress-плагин.

- **Интеграции:** AmoCRM, Битрикс24, Roistat
- **Плюсы:** готовый плагин, сильная аналитика по звонкам
- **Минусы:** платная модель, сложные кастомные интеграции требуют API

## Как выбрать

1. **РФ/СНГ аудитория** → Яндекс.Метрика + Clarity
2. **Глобальная аудитория** → GA4 + Clarity
3. **Нужны опросы и feedback** → Hotjar вместо Clarity
4. **E-commerce с рекламой** → добавьте Roistat или Calltouch
5. **Минимальный бюджет** → Clarity (бесплатно) + GA4 (бесплатно)
