## Обзор

Email-маркетинг для WordPress решает три задачи:
1. **Сбор подписчиков** — формы на сайте, pop-up, чекбоксы в комментариях и WooCommerce
2. **Рассылки** — welcome-цепочки, промо-кампании, триггерные письма
3. **Автоматизация** — поведенческие цепочки, abandoned cart, post-purchase follow-up

Все сервисы ниже интегрируются с WordPress через плагины или API.

## Российские сервисы

### UniSender

Простой российский сервис email-рассылок. Плагин **UniSender Integration** добавляет формы подписки в сайдбар, посты, страницы и футер.

- **Интеграция:** плагин → API-ключ → список рассылки. Шорткоды и виджеты для форм.
- **Плюсы:** простая интеграция «из коробки», поддержка доп. полей, хорошая доставляемость по РФ
- **Минусы:** плагин только для сбора подписчиков, рассылки — в интерфейсе UniSender
- **Цены:** бесплатный (ограничен), платные от ~500 ₽/мес

### SendPulse

Мультиканальный маркетинг: email + SMS + web push + чат-боты в мессенджерах. **Бесплатно до 15 000 писем/мес**.

- **Плагин:** SendPulse Email Marketing Newsletter — API-ключи, импорт базы из WordPress, стилизация форм
- **Плюсы:** мультиканальность, бесплатный тариф, готовые шаблоны, drag-and-drop редактор
- **Минусы:** перегруженный интерфейс, ранее были XSS-уязвимости в плагине
- **Для кого:** блоги, малый бизнес, магазины на WordPress

### Carrot Quest

Гибрид: чат-бот + email-рассылки + микро-CRM. Глубокая поведенческая персонализация: триггеры по поведению → автоматические сообщения.

- **Плагин:** WordPress + интеграция с WooCommerce (до 7.1.0)
- **Функции:** чат-окно, чат-боты, поведенческие поп-апы, триггерные email-воронки, A/B тесты по доходу
- **Минусы:** тарифы зависят от количества посетителей, жалобы на авто-списание доплат
- **Для кого:** WooCommerce-магазины, онлайн-школы, сервисные компании

## Международные сервисы

### Mailchimp

Самый популярный email-маркетинг в мире. Интеграция через официальный плагин или **MC4WP (Mailchimp for WordPress)** — pop-ups, inline-формы, чекбоксы на комментариях и WooCommerce.

- **Плюсы:** простая настройка, много шаблонов, хорошая доставляемость, огромная экосистема
- **Минусы:** дорожает с ростом базы (pricing по контактам), автоматизация слабее чем ActiveCampaign/Klaviyo
- **Цены:** Free (500 контактов) → Essentials $13/мес → Standard $20/мес → Premium $350/мес
- **Для кого:** блоги и small business

### Brevo (бывший Sendinblue)

**Pay-per-email** — платите за количество писем, а не за количество контактов. Бесплатный тариф: 9 000 писем/мес, неограниченные контакты.

- **Плагин:** официальный — формы, SMTP для транзакционных писем, WooCommerce
- **Функции:** Aura AI (subject lines, copy), abandoned cart, post-purchase follow-ups, WhatsApp
- **Плюсы:** выгодная модель оплаты, очень щедрый бесплатный тариф, SMTP для WordPress
- **Минусы:** автоматизация слабее чем ActiveCampaign/Klaviyo, UI иногда глючит
- **Для кого:** растущие проекты, которым важна цена

### HubSpot

CRM + маркетинг + sales в одном. Официальный плагин **HubSpot All-in-One Marketing**: формы, popups, live chat, чат-боты → все данные в единой CRM.

- **Плюсы:** единая база (CRM + маркетинг + sales), inbound-маркетинг «из коробки», бесплатный CRM
- **Минусы:** маркетинг-автоматизация дорого, vendor lock-in
- **Для кого:** компании, уже использующие HubSpot как CRM

## Как выбрать

1. **Блог / малый бизнес** → Mailchimp (просто) или Brevo (дёшево)
2. **РФ аудитория + простота** → UniSender
3. **Мультиканальность (email+SMS+push)** → SendPulse
4. **WooCommerce + чат-боты + CRM** → Carrot Quest
5. **Уже используете HubSpot** → HubSpot
6. **Много контактов, мало писем** → Brevo (pay-per-email)

Для транзакционных писем WordPress (сброс пароля, уведомления) все сервисы выше поддерживают SMTP. Подробнее о плагинах — [Плагины WordPress: основы выбора и установки](../../plugins/plugins-basics.md).
