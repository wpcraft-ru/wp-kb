## Обзор

No-code платформы автоматизации связывают WordPress с внешними сервисами: CRM, мессенджерами, Google Sheets, платёжными системами. Принцип работы: **триггер в WordPress → действие в другом сервисе** (или наоборот).

Типовые сценарии:
- Новый пользователь → CRM / Google Sheets
- Новая запись → рассылка в Telegram / WhatsApp
- Заказ WooCommerce → CRM + уведомление менеджеру
- Форма → Slack + email
- Google Sheets / AI → авто-пост в WordPress

## Российские сервисы

### Albato

1 000+ сервисов, визуальный конструктор «триггер-действие». Три способа подключения WordPress: Application Password, Basic Auth, JWT-токен.

- **Типовые сценарии:** новый пользователь → Google Sheets, заказ WooCommerce → CRM + уведомления, синхронизация с Tidycal, Nifty, Odoo
- **Плюсы:** простой визуальный конструктор, широкая база интеграций, работа в реальном времени, поддержка на русском
- **Минусы:** нет официального нативного плагина для WordPress

### ApiMonster

Российский no-code коннектор с фокусом на связку WordPress + CRM + 1С. Готовые бандлы: WordPress + Bitrix24, Poster, MPMonster, Контур.Школа.

- **Особенности:** универсальные веб-хук-коннекторы, ApiMonster AI (генерация контента, обработка лидов)
- **Плюсы:** 5–10 минут на подключение, не нужен программист для типовых связок
- **Минусы:** сложные бизнес-логики требуют доработки

## Международные сервисы

### Make (ex-Integromat)

Визуальный scenario builder с ветвлением, фильтрами и трансформациями данных. 1 000+ сервисов. Официальный плагин **Make Connector** для WordPress.

- **WordPress-модули:** посты, категории, пользователи, медиа, комментарии
- **Плюсы:** сложные многошаговые workflow, мощный API, дешевле Zapier при масштабировании
- **Минусы:** выше порог входа чем Zapier, плагин иногда на review
- **Цены:** Free (1 000 ops/мес) → платные тарифы

### Zapier

Самая популярная no-code платформа: 5 000+ приложений. Плагин **Zapier for WordPress** для отправки триггеров.

- **Плюсы:** очень простой интерфейс, огромная библиотека приложений, готовые шаблоны (Zaps)
- **Минусы:** дорожает с ростом задач (per task), только линейная логика
- **Цены:** Free (100 задач/мес) → Starter $20/мес → Professional $49/мес → Team $69/мес

### n8n

**Open-source self-hosted** платформа. Node-based визуальный редактор, официальный WordPress-node (CRUD: посты, страницы, пользователи).

- **Подключение:** WordPress node (username + app password) или Webhooks через WP Webhooks / Bit Integrations
- **Плюсы:** полный контроль и приватность, гибкая логика (условия, циклы, code nodes), выгодно при масштабировании
- **Минусы:** высокий порог входа, maintenance overhead (обновления, бэкапы)
- **Для кого:** технические команды, агентства, compliance-sensitive проекты

## Как выбрать

1. **Максимум готовых интеграций** → Zapier
2. **Сложные многошаговые сценарии** → Make
3. **Приватность и self-hosted** → n8n
4. **РФ: WordPress + CRM/1С** → ApiMonster
5. **РФ: универсальные связки** → Albato

Все сервисы используют REST API WordPress или webhooks. Для расширенной кастомизации см. [WP-CLI и автоматизация](../../how-to/multiple-sites-management.md).
