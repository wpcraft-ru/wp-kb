## Зачем нужен SEO-плагин

SEO-плагин для WordPress делает три вещи:
1. **Генерирует мета-теги** (title, description, Open Graph) — чтобы поисковики и соцсети понимали страницу.
2. **Строит XML-карту сайта** — дорожную карту для Google и Яндекса.
3. **Добавляет Schema-разметку** — структурированные данные для расширенных сниппетов (звёзды, цены, FAQ).

Без SEO-плагина WordPress сам генерирует только базовый title. Все остальные метаданные отсутствуют.

## Рейтинг SEO-плагинов (2025–2026)

### 🥇 The SEO Framework — самый простой и удобный

**Рейтинг:** 4.9/5 | **Установок:** 100K+

Философия — «настроил и забыл». Автоматически генерирует мета-теги, Schema, канонические URL, Open Graph и Twitter Cards. Не требует ручной настройки для каждой страницы.

**Почему лучший по удобству:**
- Почти не влияет на скорость сайта (+0.04s к загрузке)
- Потребляет всего **23.71 KB памяти** — в 12 раз меньше конкурентов
- Нет рекламы, навязчивых уведомлений, трекинга
- White-label: клиенты не видят плагин
- Автоматически блокирует AI-краулеры

**Ограничения:**
- Нет визуальных «светофоров» и оценок для редакторов
- Нет встроенного анализа читаемости
- Дополнительные функции (Local SEO, мониторинг) — через платные Extensions
- Меньше документации и community

**Кому подходит:** техническим специалистам, минималистам, сайтам где важна скорость (Core Web Vitals).

---

### 🥈 Rank Math — современный с платными опциями

**Рейтинг:** 4.9/5 | **Установок:** 3M+

Самый функциональный SEO-плагин в бесплатной версии. Запущен в 2018 году и быстро догнал старожилов.

**Бесплатно:**
- 5 фокусных ключевых слов на страницу
- SEO-оценка от 0 до 100 с цветовой индикацией
- Полноценный менеджер редиректов
- 404-мониторинг
- 18+ типов Schema (Article, Recipe, Product, FAQ, How-To, Review...)
- Социальные превью для Facebook и Twitter
- Интеграция с Google Search Console прямо в админке
- **Интеграция с Яндекс.Вебмастер** (верификация, IndexNow)
- SEO для WooCommerce

**PRO ($59–$95/год):**
- Content AI: 40+ AI-инструментов в редакторе (генерация заголовков, аутлайнов, анализ конкурентов)
- Отслеживание позиций до 1000 ключевых слов
- Instant Indexing (уникальная функция — нет у конкурентов)
- SEO-отчёты на email
- Видео-SEO

**Производительность:** +0.35 MB памяти, +0.01s к загрузке — в 4 раза легче Yoast.

**Кому подходит:** фрилансерам, агентствам, продвинутым пользователям — максимум функций бесплатно.

---

### 🥉 Yoast SEO — самый популярный, но кривоват

**Рейтинг:** 4.8/5 | **Установок:** 5M+

Стандарт индустрии с 2010 года. Самый узнаваемый SEO-плагин.

**Бесплатно:**
- 1 фокусная ключевая фраза
- Светофорная система (🟢 🟡 🔴) — понятна редакторам
- Анализ SEO и читаемости в реальном времени
- XML-карта сайта, канонические URL
- Базовая Schema (Unified Graph)
- Open Graph (превью соцсетей — только в Premium)

**Premium ($99/год):**
- Множественные ключевые слова (до 5 синонимов)
- AI-генерация заголовков и meta description
- Редиректы (менеджер перенаправлений)
- Превью соцсетей (Facebook, Twitter)
- WooCommerce SEO — отдельное расширение

**Проблемы:**
- **Самый тяжёлый:** +1.62 MB памяти, +0.18s к загрузке — в 4.6 раз больше Rank Math
- 32.4 MB размер папки, 97K строк кода
- Ключевые функции (редиректы, соцпревью) — в Premium
- Много навязчивой рекламы в админке

**Кому подходит:** контент-командам и новичкам — если скорость сайта не критична и нужны понятные подсказки.

## Остальные плагины

### AIOSEO (All in One SEO)

**Рейтинг:** 4.8/5 | **Установок:** 3M+

Лидер по интеграции с WooCommerce и локальным SEO.

**Сильные стороны:**
- **WooCommerce SEO:** автоматическая Product Schema с ценами, наличием, отзывами, доставкой и возвратами
- **Локальное SEO:** NAP (Name, Address, Phone), Google Maps, Knowledge Graph с часами работы
- Link Assistant — автоматические подсказки внутренних ссылок
- Search Statistics — данные из Google Search Console в админке

**Цены:** Basic $49 → Pro $199 → Elite $299/год.

**Кому подходит:** интернет-магазинам на WooCommerce, локальному бизнесу.

---

### SEOPress

**Рейтинг:** 4.8/5 | **Установок:** 300K+

Чистый интерфейс без рекламы и выгодная цена для агентств.

**Бесплатно:**
- **Неограниченное количество ключевых слов** для анализа
- Редиректы и 404-мониторинг
- Социальные превью
- Интеграция с Яндекс.Вебмастер

**PRO ($49–$149/год):**
- **ИИ-генерация мета-тегов** через 5 провайдеров: OpenAI, Claude, Gemini, Mistral, DeepSeek
- Массовая генерация meta title/description, OG-тегов и alt-текстов
- White-label (нет брендинга SEOPress)

**Кому подходит:** агентствам (Unlimited за $149), тем кто хочет ИИ-генерацию контента.

## Производительность: цифры

На основе независимых тестов WP Rocket на идентичных staging-окружениях:

| Плагин | Память | Влияние на загрузку | Размер |
|--------|--------|---------------------|--------|
| The SEO Framework | 23.71 KB | +0.04s | ~3 MB |
| Rank Math | +0.35 MB | +0.01s | 11.8 MB |
| SEOPress | Минимальное | Минимальное | ~8 MB |
| AIOSEO | Умеренное | Умеренное | ~25 MB |

> На бюджетном хостинге разница становится критичной. Yoast прибавляет 0.18s — на слабом сервере это может ухудшить Core Web Vitals.

## Миграция между SEO-плагинами

Все современные SEO-плагины поддерживают импорт метаданных из конкурентов — смена плагина **не теряет SEO-данные**.

### Базовый алгоритм

1. **Сделайте полный бэкап** сайта
2. Установите новый плагин **при активном старом**
3. Запустите инструмент импорта в новом плагине
4. Проверьте импортированные данные (title, description, редиректы)
5. Деактивируйте старый плагин

> ⚠️ **Не держите два SEO-плагина активными одновременно.** Это создаёт дублирование метаданных и конфликтующую Schema.

### Популярные маршруты

| Из → В | Инструмент |
|--------|-----------|
| Yoast → Rank Math | Setup Wizard → Import from Yoast |
| Yoast → SEOPress | SEOPress → Import/Export → Import from Yoast |
| AIOSEO → Rank Math | Setup Wizard → Import from AIOSEO |
| Любой → Rank Math | Advanced Settings → Import & Export |

## Интеграция с Яндексом

Для сайтов, ориентированных на русскоязычную аудиторию, критична интеграция с Яндекс.Вебмастер.

**Лучшие плагины для Яндекса:**
- **Rank Math** — верификация, sitemap, IndexNow
- **SEOPress** — верификация, синхронизация данных Вебмастера
- **Yoast** — только базовая верификация

The SEO Framework и AIOSEO не предоставляют встроенной интеграции с Яндексом — верификацию придётся делать вручную через мета-тег или HTML-файл.

## Site Kit for Yandex — панель интеграции с Яндекс для РФ

Site Kit for Yandex ([wpcraft.ru/projects/site-kit-for-yandex](https://wpcraft.ru/projects/site-kit-for-yandex)) — **комплементарный** инструмент для сайтов в РФ. Это не замена SEO-плагину, а дополнение к нему.

### Что он делает

Объединяет сервисы Яндекса в единой панели в админке WordPress:

| Сервис | Назначение |
|--------|-----------|
| Яндекс Метрика | Веб-аналитика, посещаемость, поведение пользователей |
| Яндекс Вебмастер | Контроль индексации, позиции в поиске, технические ошибки |
| IndexNow | Мгновенное уведомление о новых/обновлённых страницах |
| Яндекс ID | OAuth-авторизация через учётную запись Яндекса |
| Яндекс РСЯ | Размещение рекламных блоков (заявлено) |
| Yandex Object Storage | Облачное хранилище (заявлено) |

### Ключевые возможности

- **Yandex Overview** — сводная страница с показателями сайта
- **URL Inspector** — проверка статуса индексации конкретных URL
- Данные Метрики прямо в админке
- Автоматическая отправка страниц на индексацию через IndexNow

### Чем отличается от SEO-плагинов

Site Kit for Yandex **не управляет мета-тегами, Schema, sitemap и редиректами**. Он показывает аналитику и статус индексации из экосистемы Яндекса. Работает в паре с любым SEO-плагином.

| Функция | Site Kit for Yandex | SEO-плагины |
|---------|:-:|:-:|
| Мета-теги | ❌ | ✅ |
| Schema | ❌ | ✅ |
| Sitemap | ❌ | ✅ |
| Редиректы | ❌ | ✅ |
| Яндекс.Метрика | ✅ | Частично |
| Панель Вебмастера | ✅ | Только верификация |
| URL Inspector | ✅ | ❌ |
| Реклама РСЯ | ✅ | ❌ |

### Для кого

Владельцы WordPress-сайтов в РФ, которым нужен единый дашборд сервисов Яндекса **в дополнение** к SEO-плагину (Rank Math, SEOPress, Yoast или другому).
