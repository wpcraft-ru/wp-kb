## Начало работы

WooCommerce — бесплатный плагин для создания интернет-магазина на WordPress. После установки:

1. Перейдите в WooCommerce → Настройки → запустите мастер установки
2. Укажите страну, валюту, способы доставки и оплаты
3. Настройте налоги (НДС/без НДС в зависимости от юрисдикции)

## Тема для магазина

WooCommerce работает с любой темой, но рекомендуется использовать WooCommerce-совместимые темы:
- **Storefront** — официальная бесплатная тема от WooCommerce
- Блочные темы из каталога WordPress.org с поддержкой WooCommerce

Для редактирования страниц магазина используйте WooCommerce-блоки в редакторе: товары, корзина, оформление заказа, фильтры.

## Добавление товаров

Товары → Добавить новый. Основные типы товаров:

| Тип | Назначение |
|---|---|
| **Простой товар** | Обычный товар с одной ценой |
| **Вариативный товар** | Товар с вариантами (размер, цвет), каждый со своей ценой и остатком |
| **Виртуальный товар** | Услуги, не требующие доставки |
| **Цифровой (загружаемый)** | Файлы: PDF, аудио, видео, ПО |
| **Внешний/партнёрский** | Ссылка на товар на другом сайте |

## Оплата

WooCommerce поддерживает множество платёжных шлюзов:

- **Прямые платежи:** Robokassa, YooKassa (ЮMoney), CloudPayments, Tinkoff
- **Онлайн-кассы:** для РФ обязательна интеграция с кассой (54-ФЗ)
- **Наложенный платёж, банковский перевод**

## Доставка

Настройка зон доставки (WooCommerce → Настройки → Доставка):

1. Создайте зоны по географическому признаку
2. Добавьте методы доставки в каждую зону:
   - **Фиксированная ставка** — задаётся вручную
   - **Бесплатная доставка** — при сумме заказа или по купону
   - **Самовывоз** — покупатель забирает заказ сам
3. Для РФ: Почта России (есть готовые плагины расчёта стоимости)

## Налоги

- Для РФ: WooCommerce по умолчанию не включает НДС в цену для B2C. Настройте:
  - WooCommerce → Настройки → Налог → включить
  - Настроить ставки (20%, 10%, 0%, без НДС)
- Проверьте соответствие требованиям вашей юрисдикции

## Управление заказами

Заказы отображаются в WooCommerce → Заказы. Возможности:
- Обновление статусов (обработка → отправлен → выполнен)
- Ручное создание заказов для офлайн-продаж
- Отправка счёт-фактур

## Аналитика

Встроенная аналитика WooCommerce (Отчёты): продажи, товары, категории, купоны. Для расширенной аналитики подключите Google Analytics через Site Kit by Google.

## Маркетинговые инструменты

- **Купоны:** WooCommerce → Купоны — скидки по проценту, сумме, условиям
- **Email-маркетинг:** интеграция с MailPoet, Unisender
- **Социальные сети:** плагины для интеграции с VK, Telegram

## PHP пример: минимальная сумма заказа

```php
add_action('woocommerce_checkout_process', function () {
    $minimum = 500;
    if (WC()->cart->subtotal < $minimum) {
        wc_add_notice(
            sprintf('Минимальная сумма заказа: %s ₽', $minimum),
            'error'
        );
    }
});
```

Что делает код:
- Не даёт оформить заказ, если сумма корзины меньше 500 ₽.
