Раздел охватывает полный цикл создания интернет-магазина на WooCommerce для self-hosted WordPress.

## Что такое WooCommerce

WooCommerce — это opensource-плагин для WordPress, который превращает сайт в полноценный интернет-магазин. Устанавливается бесплатно из [репозитория WordPress.org](https://wordpress.org/plugins/woocommerce/). Работает одинаково на WordPress.com и self-hosted WordPress.org.
