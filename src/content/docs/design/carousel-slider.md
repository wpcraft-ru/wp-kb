## Зачем нужны карусели

Веб-карусели — элегантный способ сгруппировать связанный контент в одном пространстве: изображения продуктов, отзывы, баннеры акций.

## Плагины для каруселей

### MetaSlider
Традиционный слайдер с множеством настроек:
- Мобильная адаптация
- Расширенные настройки
- Настройки доступности
- Режим разработчика

### Smart Slider 3
Мощный конструктор слайдеров с визуальным редактором.

### Carousel Slider Block for Gutenberg
Блочный слайдер, поддерживает другие блоки внутри слайда и WooCommerce.

## Carousel Slider Block for Gutenberg

### Установка и использование
1. Установите и активируйте плагин
2. В редакторе: `/slider` → выбрать Carousel Slider
3. Добавить блоки внутри (например, 3 Image блока)
4. Выделить родительский блок → alignment: wide width

### Настройки
| Параметр | Описание |
|---|---|
| Количество слайдов к показу | Сколько слайдов видно одновременно |
| Слайдов за прокрутку | Сколько проматывается за раз |
| Скорость анимации | Время перехода |
| Стрелки и точки | Навигация (вкл. по умолчанию) |
| Responsive breakpoints | Сколько слайдов на каждом размере экрана |
| Aspect ratio | Единый размер изображений |

### За пределами изображений
Внутрь слайдов можно добавлять Cover блоки с Heading + Spacer + Buttons. Установите «Slides to show: 1» для полноэкранного слайдера.

## Доступность и слайдеры

Слайдеры могут создавать проблемы с доступностью. Минимальные требования:

- **Alt text** для всех изображений
- **Управление с клавиатуры** — переключение слайдов без мыши
- **Поддержка скринридеров** — aria-метки, роли
- **Избегайте автопрокрутки** — дайте пользователю контроль
- Если auto-play необходим — обязательна **кнопка паузы**
- **RTL** — поддержка языков с написанием справа налево (WordPress поддерживает из коробки)
