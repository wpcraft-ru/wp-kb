## Почему WordPress — платформа для дизайнеров

WordPress — не просто CMS, а полноценная дизайн-среда. С блочным редактором, библиотеками паттернов и Site Editor вы можете создавать визуально сложные сайты **без написания кода**.

### Ключевые инструменты

- **Block Editor (Gutenberg)** — вставка и настройка блоков (параграфы, изображения, кнопки, галереи) через визуальный редактор. Каждый блок — самодостаточный элемент с собственными настройками стилей.
- **Site Editor** — редактирование всего сайта: от шапки до подвала. Доступен для блочных тем через **Внешний вид → Редактор сайта**.
- **Style Book** — цифровой style guide, показывающий, как каждый блок выглядит в контексте всех остальных. Идеально для проектирования без готового контента.
- **Global Styles** — единые настройки цветов, шрифтов и макета, применяемые ко всему сайту. Меняете один раз — обновляется везде.
- **Patterns (паттерны)** — готовые наборы блоков для быстрой сборки страниц. Можно создавать свои синхронизированные и несинхронизированные паттерны.

### Работа с медиа

- **Cover block** — изображение или видео с наложенным текстом. Используется для шапок, баннеров, разделителей секций.
- **Gallery block** — автоматическая галерея изображений с настройкой колонок, стилей и lightbox.
- **Media & Text** — изображение и текст бок о бок. Идеально для «о нас», «услуги», «контакты».
- **Query Loop block** — мощная выборка контента для новостных сайтов, журналов, блогов.

### Расширение через плагины

- **Формы** — контактные формы, опросы
- **Продвинутые галереи и слайдеры** — MetaSlider, Smart Slider 3, Carousel Slider Block
- **Grid block** — создание сложных сеток

WordPress — это open-source платформа, которая работает на 40%+ веба. Дизайн, созданный здесь, принадлежит вам, без привязки к проприетарным сервисам.
