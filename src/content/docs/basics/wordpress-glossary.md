## Основные термины

### WordPress и его версии
- **WordPress** — бесплатная open-source CMS для публикации контента в интернете.
- **WordPress.org** — сайт проекта WordPress, где можно скачать дистрибутив, плагины и темы.
- **WordPress.com** — коммерческий сервис managed-хостинга от Automattic на базе WordPress.

### Адрес и домен
- **Домен (domain name)** — адрес сайта в интернете (например, `mysite.ru`).
- **URL (веб-адрес)** — полный путь к ресурсу, включая протокол (например, `https://mysite.ru/about`).
- **DNS-запись** — настройка, определяющая как домен работает (A, CNAME, MX и т.д.).
- **Регистратор доменов** — компания, регистрирующая домены (nic.ru, reg.ru).
- **SSL-сертификат** — обеспечивает HTTPS и безопасное соединение (Let's Encrypt — бесплатный вариант).

### Хостинг
- **Хостинг (web hosting)** — место хранения файлов сайта на сервере.
- **Self-hosted** — вы самостоятельно устанавливаете и обслуживаете WordPress на своём хостинге.
- **Managed WordPress** — хостинг с автообновлениями, бэкапами и безопасностью (WordPress.com, SiteGround).

### Админка
- **WP Admin (wp-admin)** — классическая панель управления WordPress, доступна по адресу `site.ru/wp-admin`.
- **Dashboard** — главный экран админки с виджетами (At a Glance, Activity, Quick Draft).
- **Admin Bar (Toolbar)** — чёрная полоса вверху экрана при входе в систему.
- **Screen Options** — настройка отображения колонок и элементов на экранах админки.

### Редактор и блоки
- **Блок (block)** — структурный элемент контента в редакторе: текст, изображение, видео, колонки.
- **Редактор блоков (Block Editor / Gutenberg)** — современный редактор WordPress, где всё строится из блоков.
- **Site Editor** — редактор для глобальных элементов сайта: шаблонов, меню, header/footer.
- **Full Site Editing (FSE)** — возможность редактировать весь сайт через блоки (в блочных темах).
- **Classic Editor** — старый редактор WordPress, основанный на TinyMCE.

### Контент
- **Запись (Post)** — динамический контент с датой, отображается в обратном хронологическом порядке.
- **Страница (Page)** — статический контент (About, Contact), не привязана к дате.
- **Рубрика (Category)** — способ группировки записей по темам.
- **Метка (Tag)** — дополнительная маркировка записей, более гибкая чем рубрики.
- **Таксономия (Taxonomy)** — система классификации контента (рубрики и метки — встроенные таксономии).

### Внешний вид
- **Тема (Theme)** — определяет визуальный дизайн сайта: цвета, шрифты, расположение элементов.
- **Дочерняя тема (Child Theme)** — расширяет родительскую тему, не затрагивая её файлы.
- **Блочная тема (Block Theme)** — тема, полностью основанная на блоках (FSE).
- **Классическая тема (Classic Theme)** — тема на PHP-шаблонах.
- **Customizer** — инструмент настройки внешнего вида с живым превью.
- **Шаблон (Template)** — управляет расположением элементов на странице или группе страниц.
- **Меню (Navigation Menu)** — список ссылок для навигации по сайту.
- **Виджет (Widget)** — модульный блок контента для sidebar или footer.

### Плагины и расширения
- **Плагин (Plugin)** — дополнение, добавляющее функциональность сайту.
- **WooCommerce** — плагин для создания интернет-магазинов.
- **Jetpack** — плагин от Automattic с набором функций: статистика, безопасность, производительность.

### Производительность
- **Кэш (Cache)** — сохранённая копия данных для ускорения загрузки.
- **CDN (Content Delivery Network)** — сеть серверов для быстрой доставки контента по всему миру.
- **SEO (Search Engine Optimization)** — оптимизация сайта для поисковых систем.

### Разработка
- **Хук (Hook)** — точка расширения в WordPress (actions и filters).
- **Action** — позволяет выполнить код в определённый момент.
- **Filter** — позволяет изменить данные перед выводом.
- **Шорткод (Shortcode)** — короткий код `[example]` для вставки динамического контента.
- **WP-CLI** — интерфейс командной строки для управления WordPress.
- **SFTP/SSH** — протоколы для безопасного доступа к файлам сервера.

### WordPress.com (справочно)
- **Automattic** — компания-создатель WordPress.com, WooCommerce, Jetpack.
- **План (Plan)** — тарифный пакет на WordPress.com с набором функций.
- **Подписка (Subscription)** — может означать тарифный план или подписку посетителей на обновления.
