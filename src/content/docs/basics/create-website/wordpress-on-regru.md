## REG.RU для WordPress

REG.RU — крупнейший российский регистратор доменов и хостинг-провайдер. WordPress можно установить автоматически через автоустановщик или вручную — инструкции для трёх панелей управления: **ispmanager**, **cPanel** и **Plesk**.

## Способ 1: Автоматическая установка

Самый простой способ — автоустановщик в панели хостинга. Доступен на всех тарифах виртуального хостинга Linux.

1. В панели управления перейдите в раздел **CMS** → **Автоматическая установка CMS**
2. Выберите **WordPress** → **Установить**
3. Укажите домен, логин и пароль администратора
4. Нажмите **Установить**

Автоустановка также доступна при заказе нового домена с хостингом или при заказе тарифа [WordPress hosting](https://www.reg.ru/hosting/wordpress). Есть бесплатный тестовый хостинг.

## Способ 2: Ручная установка

Универсальный метод — подходит для любой панели управления. Состоит из трёх шагов.

### Шаг 1: Создание базы данных

#### ispmanager

1. Раздел **Базы данных** → **Создать базу данных**
2. Укажите имя БД и выберите пользователя (или создайте нового)
3. Сгенерируйте пароль → **Ок**

> Ограничение ispmanager: к названию БД добавляется префикс `u1234567_` (9 символов), максимум имени — 16 символов. Вводимое имя не должно превышать 7 символов.

#### cPanel

1. Раздел **Базы данных** → **Помощник баз данных MySQL**
2. Укажите название БД → **Следующий шаг**
3. Укажите имя пользователя и пароль → **Создать пользователя**
4. Отметьте все права → **Следующий шаг**

> Если интерфейс отличается — переключите тему с `paper_lantern` на `jupiter` в разделе «Основная информация».

#### Plesk

1. Раздел **Базы данных** → **Добавить базу данных**
2. Заполните: имя БД, сайт, имя пользователя, пароль
3. Поставьте галочку «разрешить доступ ко всем БД» → **ОК**

### Шаг 2: Загрузка WordPress

1. Скачайте актуальный WordPress с [ru.wordpress.org](https://ru.wordpress.org/)
2. В панели управления откройте файловый менеджер сайта
3. Удалите все файлы из корневой папки (кроме `webstat` в ispmanager, `cgi-bin` в cPanel/Plesk)
4. Загрузите ZIP-архив WordPress и извлеките его
5. Переместите все файлы из папки `wordpress/` в корень сайта

### Шаг 3: Установка через install.php

1. Откройте ваш домен в браузере
2. На странице установки нажмите **Вперёд**
3. Заполните реквизиты БД из шага 1:
   - **Сервер БД** — `localhost`
   - **Имя БД**, **Пользователь**, **Пароль** — из шага 1
   - **Префикс таблиц** — рекомендуется сменить `wp_` на уникальный (буквы + цифры)
4. Нажмите **Запустить установку**
5. Заполните данные сайта:
   - Название сайта
   - Логин администратора (для `wp-admin`)
   - Пароль
   - Email администратора (для восстановления пароля)
6. Нажмите **Установить WordPress**

Готово! Вход в админку: `ваш-домен.ru/wp-admin`.
