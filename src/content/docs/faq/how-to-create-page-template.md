## Краткий ответ

Шаблон страницы можно создать двумя способами: визуально через Site Editor (для блочных тем) — без кода, прямо в админке, или через PHP-файл в child theme (для классических тем). Современные проекты используют Site Editor.

## Способ 1: Site Editor (блочные темы) — рекомендуется

1. **Внешний вид → Редактор сайта → Шаблоны**
2. Нажать **+** («Добавить шаблон»)
3. Выбрать тип контента: страница, запись, архив
4. Дать имя (например, «Без заголовка»)
5. Редактировать блоками — изменить расположение заголовка, добавить баннер, настроить ширину

Где выбрать: в редакторе страницы → боковая панель → «Шаблон».

Изменения сохраняются в БД и не теряются при обновлении темы. Подробнее: [Шаблоны WordPress](../themes/templates.md), [Site Editor](../themes/site-editor.md).

## Способ 2: PHP-шаблон (классические темы)

1. Создайте файл в child theme: `page-landing.php`
2. Добавьте заголовок:

```php
<?php
/*
 * Template Name: Мой шаблон
 * Template Post Type: page
 */

get_header();
// ваша разметка
get_footer();
```

3. В редакторе страницы → «Атрибуты страницы» → «Шаблон» → выбрать «Мой шаблон»

Подробнее о child theme: [Темы и Customizer](../themes/themes-child-themes-customizer.md). О типах тем: [Типы тем](../themes/theme-types.md).

## Какой способ выбрать

| | Site Editor | PHP-шаблон |
|---|---|---|
| Тип темы | Блочная | Классическая |
| Сложность | Низкая (без кода) | Средняя (PHP) |
| Сохранность | БД (переживает обновления) | Файл в child theme |

> Для новых сайтов выбирайте блочную тему и Site Editor — это будущее WordPress. [Типы тем](../themes/theme-types.md)
