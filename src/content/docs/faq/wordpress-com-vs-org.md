## Суть различия

WordPress — это бесплатное open-source ПО (CMS). И WordPress.com, и WordPress.org используют один и тот же код WordPress. Разница — в модели хостинга и владения:

- **WordPress.com** — коммерческий сервис от Automattic с managed-хостингом. Вы платите за план — получаете хостинг, автообновления, бэкапы и поддержку.
- **WordPress.org / self-hosted** — вы скачиваете WordPress бесплатно и устанавливаете на свой хостинг. Вы полностью контролируете сайт, но и отвечаете за всё сами.

## Таблица сравнения

| Критерий | WordPress.com | WordPress.org (self-hosted) |
|---|---|---|
| **Стоимость** | Бесплатный тариф с ограничениями; Personal, Premium, Business, Commerce ($4-45+/мес) | WordPress бесплатен. Расходы: хостинг ($3-30/мес), домен (~$10/год) |
| **Хостинг** | Managed — включён в план | Выбираете и оплачиваете сами (shared, VPS, облачный) |
| **Безопасность** | Встроенная защита, автообновления, SSL из коробки | Ваша ответственность: настроить SSL, следить за обновлениями, установить security-плагины |
| **Резервное копирование** | Автоматическое (на Business и выше — можно восстановить) | Настроить самостоятельно: плагины (UpdraftPlus, BackWPup) или бэкапы хостинга |
| **Плагины** | На платных планах — можно устанавливать любые (50,000+) | Полная свобода: любые плагины из репозитория .org |
| **Темы** | Встроенный каталог + загрузка своих тем на платных планах | Полная свобода: любые темы |
| **Поддержка** | 24/7 поддержка на платных планах, форумы на бесплатном | Форумы WordPress.org, документация, саппорт хостинга/плагинов |
| **CDN и скорость** | Встроенный CDN, безлимитный трафик | Зависит от хостинга; можно подключить Cloudflare |
| **Домен** | Бесплатный поддомен `.wordpress.com` или свой домен | Только свой домен (покупается отдельно) |
| **Монетизация** | WordAds (реклама) на Premium+, своя реклама на Business+ | Полная свобода: любые рекламные сети |
| **Доступ к коду** | SFTP/SSH, WP-CLI, GitHub Deployments на Business+ | Полный доступ через хостинг всегда |
| **Владение контентом** | Вы владеете контентом, можете экспортировать | Вы владеете всем |

## Что выбрать для self-hosted

Если вы выбрали WordPress.org (self-hosted), вам потребуется:

1. **Хостинг** — shared (TimeWeb, Beget) для старта, VPS (VDSina, FirstVDS) для серьёзных проектов
2. **Домен** — регистрация у регистратора (nic.ru, reg.ru) или через хостинг
3. **SSL-сертификат** — бесплатный Let's Encrypt (обычно включён в хостинг)
4. **Бэкапы** — плагин UpdraftPlus или встроенные бэкапы хостинга
5. **Безопасность** — плагин Wordfence или Solid Security

## Когда self-hosted обязателен

- Нужна полная свобода в коде (кастомные PHP, правки ядра)
- Специфические требования к серверному окружению
- Интеграция с внутренними системами компании
- Мультисайт (WordPress Multisite)
- Юридические требования к размещению данных (152-ФЗ — серверы в РФ)

## ⚠️ Функции только для WordPress.com

Эти функции недоступны в self-hosted WordPress и не имеют прямых аналогов:

- **AI Website Builder** — генерация сайта через AI-промпты
- **Express Website Design Service** — создание сайта под ключ командой .com
- **Onboarding Sessions** — персональные видеозвонки со специалистом
- **Happiness Engineers** — выделенная техподдержка 24/7

Для self-hosted альтернативы: найм фрилансера/агентства, форумы WordPress.org, платные AI-плагины (AI Engine, Bertha AI).
